# WPS 灵犀自定义技能包

为体育学研究者定制的 AI 技能集合，覆盖中国体育学期刊投稿、社会学研究方法、质性分析、文献综述写作和中文学术降 AIGC 检测率。

## 包含技能

| 技能 | 说明 |
|------|------|
| Chinese-Sport-Science-Journal-Skills | 12 本 CSSCI 体育学期刊投稿指引 |
| Sociological-Studies-Skills | 《社会学研究》期刊全流程投稿指引 |
| thematic-analysis | Braun & Clarke 六阶段主题分析 |
| chinese-de-aigc | 中文学术论文降 AIGC 检测率 |
| literature-review | 五步法文献综述写作 |

## 安装方式

将对应目录复制到 WPS 灵犀的 `skills/` 目录下即可使用。
