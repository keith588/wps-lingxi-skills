# 股票分析

基于实时市场数据的全面证券分析工具，覆盖 **A股、港股、美股、ETF、基金**等所有品种，通过浏览器技能访问金融网站获取数据，**不依赖任何外部 API**。

---

> **⚠️ 使用前必读**
>
> 本技能依赖 **browser 技能** 驱动浏览器操作。如果你尚未阅读 browser 技能，**必须先阅读 `skills/browser/SKILL.md`**，了解浏览器初始化方式与 `b.navigate`、`b.click` 等方法的用法，再执行本技能中的任何步骤。

---

## 核心能力

- **公司研究**：获取公司简介、业务信息、高管团队
- **技术分析**：访问价格走势图、技术指标、市场展望
- **基本面分析**：查看估值数据、分析师评级
- **内部人员动态**：追踪内部人员持股与交易记录
- **监管文件**：访问 SEC 申报历史及相关文件（美股）
- **多股票对比**：依次访问各股票页面，汇总并排对比

---

## 数据获取方式

所有数据均通过 **browser 技能** 从以下网站实时读取。

### 数据源与 URL 规则

| 数据类型 | 数据源 | URL 模板 |
| -------- | ------ | -------- |
| 美股行情 / 公司概况 | 东方财富网（美股） | `https://quote.eastmoney.com/us/{SYMBOL}.html` |
| 美股搜索 | 东方财富网（美股） | `https://so.eastmoney.com/us/result?key={SYMBOL}` |
| SEC 申报文件（10-K / 10-Q / 8-K） | SEC EDGAR | `https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK={SYMBOL}&type={TYPE}&dateb=&owner=include&count=10` |
| A 股搜索 | 东方财富网 | `https://so.eastmoney.com/cn/result?key={股票名称或代码}` |
| A 股详情 | 东方财富网 | `https://quote.eastmoney.com/{6位股票代码}.html` |
| 基金净值 / 排行 | 天天基金网 | `https://fund.eastmoney.com/` |

> `{SYMBOL}` 替换为股票代码（如 `AAPL`、`TSLA`），`{TYPE}` 替换为申报类型（如 `10-K`）。

## 常用工作流程

### 1. 美股公司概览 → 深度研究

```
用户："告诉我关于 AAPL 的信息"
→ b.navigate("https://quote.eastmoney.com/us/AAPL.html")
  提取：当前价格、涨跌幅、市值、PE、成交量、52 周高低
→ 在页面中寻找财务摘要、分析师评级等板块，点击展开获取更多数据
```

### 2. 美股技术分析

```
用户："TSLA 现在值得买入吗？"
→ b.navigate("https://quote.eastmoney.com/us/TSLA.html")
  提取：当前价格、52 周高低、成交量、市盈率、历史走势图
→ 根据页面内容提取技术面信息，结合估值给出综合判断
```

### 3. 尽职调查套餐（美股）

```
用户："对 MSFT 进行全面分析"
→ b.navigate("https://quote.eastmoney.com/us/MSFT.html")
  提取：行情、市值、基本面数据
→ b.navigate("https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=MSFT&type=10-K&dateb=&owner=include&count=10")
  提取：最新年报申报列表，点击对应链接查看详情
```

### 4. 多股票对比

```
用户："对比 AAPL、MSFT 和 GOOGL"
→ 依次 b.navigate 三只股票的东方财富美股页面
→ 整理关键指标（市盈率、市值、涨跌幅、52周高低）并排输出对比表格
```

### 5. 行业研究

```
用户："分析科技股：AAPL、NVDA、AMD"
→ 依次访问各股票东方财富美股页面
→ 汇总各股核心指标，输出行业横向对比与综合研判
```

### 6. A 股查询

```
用户："查一下贵州茅台的股价"
→ b.navigate("https://so.eastmoney.com/cn/result?key=贵州茅台")
  在搜索结果中找到对应股票并点击进入详情页
→ 提取实时行情、涨跌幅、换手率、市盈率等数据
```

### 7. SEC 申报文件查询

```
用户："查看 NVDA 的 10-K 年报"
→ b.navigate("https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=NVDA&type=10-K&dateb=&owner=include&count=10")
  提取：申报日期、标题列表
→ 点击具体申报记录进入详情页，获取完整文件及附件链接
```

## 关键数据说明

### 东方财富美股页面可提取数据

| 数据项 | 说明 |
| ------ | ---- |
| 当前价格 / 涨跌幅 | 实时行情 |
| 市值 | 总市值 |
| PE（市盈率） | 估值参考 |
| 52 周高低 | 价格区间 |
| 成交量 | 市场活跃度 |
| 历史走势图 | 可通过页面交互切换时间范围 |

### SEC EDGAR 申报类型

| 参数值 | 文件类型 |
| ------ | -------- |
| `10-K` | 年度报告 |
| `10-Q` | 季度报告 |
| `8-K` | 重大事件披露 |
| `DEF 14A` | 股东大会委托书 |

## 何时使用本技能

**当用户提及以下内容时，启动浏览器获取数据：**

- **股票代码**："AAPL"、"TSLA"、"$MSFT"、"股价"、"股票信息"
- **分析请求**："分析"、"研究"、"深入了解"、"告诉我关于 [股票] 的信息"
- **对比需求**："对比"、"vs"、"versus"、"哪个更好"
- **价格查询**："价格"、"走势图"、"表现"、"趋势"、"涨跌"
- **申报文件**："SEC 文件"、"10-K"、"10-Q"、"财报"、"财务报表"
- **公司信息**："[公司] 是做什么的"、"关于 [公司]"

**必要的浏览器访问组合：**

- 美股行情 → 东方财富美股详情页
- 美股搜索（不知道具体代码）→ 东方财富美股搜索页
- SEC 文件 → SEC EDGAR 对应申报类型页面
- A 股 → 东方财富 A 股搜索后进入详情页
- 多只股票对比 → 依次访问各股票页面，汇总成表格

## 最佳实践

1. **先读 browser 技能** — 若未阅读 `skills/browser/SKILL.md`，必须先读后再操作
2. **先宏观后微观** — 先访问行情主页获取全貌，再按需点击展开子板块
3. **逐一提取再汇总** — 多股票对比时依次获取数据，最终整理成结构化对比表格
4. **区分市场** — A 股使用东方财富 A 股页面，美股使用东方财富美股页面，SEC 文件用 EDGAR
5. **善用页面交互** — 走势图可通过点击切换时间范围（1个月/3个月/1年等）
6. **搜索兜底** — 若直接访问详情页无结果，先用搜索页确认股票代码后再访问
