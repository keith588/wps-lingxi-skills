# 告知 · 汇报

> **适用目的**：让受众准确、快速地获取事实与结论——不追求被说服，但求"扫一眼就懂、记得住要点"。
> **典型场景**：工作汇报、项目/季度总结、绩效述职、政务与国企报告、行业研报的事实呈现、学术与科研报告/答辩。
> **何时选它**：受众是来"了解情况、核对进展、获取数据"的，而非来被你打动或学一门技能的。

## 设计理念

- **结论先行**：每页先给结论或结果，再展开过程与数据；受众扫标题即可抓到本页要点。
- **标题即观点**：页面标题写成一句结论（如"Q3 营收同比 +23%，超目标 8 个百分点"），而非"营收情况"这类无信息量的主题词。
- **事实驱动**：用数据和客观成果搭叙事骨架；以"提升 23%"取代"显著提升"，避免主观修辞与模糊措辞。
- **可追溯**：关键数据与外部引用注明来源；涉及财务时交代口径（含税/不含税、合并/单体）。
- **可视优先**：能用图表不用文字描述，能用结构图不用纯列表。

## 叙事与结构

- 整体按金字塔结构组织：顶层结论 -> 分组支撑论点（彼此互斥、合起来穷尽）-> 颗粒数据。
- 单页内部按"结论 -> 证据 -> 启示"或"成果 -> 过程 -> 下一步"展开。
- 每页标题做"单点测试"：一句话只讲一个观点；标题里出现"和/并"，多半该拆成两页。
- 提出问题或挑战时，必须配上对策或行动计划；只摆问题不给方向不可接受。
- 语气客观，既不过度自谦也不自我宣传。

## 受众与信息密度

- 密度：中高，页面充实但不拥挤；每页聚焦一个主题并给足支撑数据。
- 受众预期"快速翻阅时秒懂结论、深读时找得到细节"——据此分层：核心结论用大字/加粗，支撑数据次级，来源注释最小。
- 每组要点控制在 3-5 条；避免长段落和纯文字墙。
- 数据成果页至少含一张图表；只罗列数字不可接受。
- 适度留白保呼吸感，但不出现大片空白或仅 2-3 条要点的稀疏页。

## 视觉与版式

- 基调：理性、克制、专业可信，而非视觉炫技。
- 满版利用空间，结论式标题横贯页顶，灰阶为主、单一强调色点睛，来源注释固定页脚且允许密集排布。
- CRAP 四原则：对比（用大小/颜色/字重制造差异，让重点跳出）· 重复（统一的视觉风格贯穿全篇）· 对齐（每个元素与其他元素有明确的视觉关联，禁止随意摆放）· 邻近（相关内容靠拢成组，无关内容用空白隔开）。
- 网格系统：用隐含栅格统一安排标题、正文、图表的层级、比例与对齐；根据内容灵活选用单栏/多栏/模块化/层级网格。网格是辅助而非束缚。
- 内容页以浅色背景为主；深色背景只用于封面、章节页、强调页。
- 优先用结构图（流程图/层级图/循环图/关系图/矩阵）表达流程、机制、组织关系，而非纯文字列表。
- 减少装饰性大图；视觉重心放在数据图与结构图上。
- 同一份文档内不同页避免完全相同的布局，靠布局变化维持阅读节奏。常见布局：单栏纵向叙述、左图右析、上图下结、双图对比、数据表 + 结论。

## 配色与字体

- 配色稳重专业；有品牌/机构色时优先取其标准色为主色。
- 三色体系：主色 + 中性灰阶 + 一种强调色；全文（含灰阶）不超过 4 种颜色。
- 强调色克制使用，只点关键数据与重点标注，使其成为视觉焦点。严禁高饱和多色拼杂、渐变滥用、荧光色。
- 字体：标题用无衬线加粗，正文用无衬线常规，保证投影/屏幕可读。
- 字号层级：封面标题 40-52px -> 页面标题 30-38px -> 副标题 24-28px -> 正文 22-28px -> 脚注/来源 12-16px；KPI 大数字 ≥36px 并用强调色。
- 正文字号下限 22px，且随内容块数量上调：内容块 ≤2 个正文 ≥32px；3 个 ≥28px；4 个 ≥26px；≥5 个 ≥22px。不要默认压到下限，先用字号和间距撑满空间。

## 数据与图表表达

- **只放必要数据**：每页只放支撑本页结论的数据，不堆砌所有放得下的数字。
- **引导视觉焦点**：用大小、颜色、位置等前注意属性把视线引向关键数据点，减少受众的认知负荷。
- **先定信息，再选图表**：图表形式由"要表达的结论"决定，而非由数据类型决定（要比大小->柱状/条形；看趋势->折线；看占比->饼/环/堆叠；看相关->散点/气泡；看流程结构或系统->结构图；看分布->直方图）。
- **精简图表元素**：删除不增进理解的视觉噪声（多余网格线、无意义颜色、3D 效果、次坐标轴、冗余刻度），最大化数据墨水比，清除 chartjunk。
- **图表配解读**：每张图附文字解读或关键数据提取，不脱离结论独立存在。
- **KPI 大数字**：核心指标 = 大数字 + 单位 + 对比基准（同比/环比/完成率/行业均值）+ 一句说明，给数字一个参照框架。
- **进度/完成率**：用进度条形状直观展示目标达成，标注百分比。
- **对比表格**：计划 vs 实际、本期 vs 上期；表头深色白字，数据行交替浅色，达成/最佳值加粗。
- **瀑布图（桥图）**：展示各驱动因素如何累加为总额变化，适合利润桥、成本拆解、同比变化量分析。
- **Marimekko 图**：列宽表总量、列高表占比，一图呈现"规模 × 份额"等双维信息；类别过多时可读性下降，非必要时改用堆叠柱或簇状柱。
- **变化量/方差图**：突出增减幅度而非绝对值，适合计划 vs 实际、本期 vs 上期的差异分析。
- **来源注释**：图表与关键数据附"来源：机构，年份"，用超链接指向原始报告，10-12px 灰色。

## 硬性承诺（不可妥协）

1. 每页标题是结论句，不是主题词。
2. 每个数据成果页至少一张图表，且图表服务于标题结论。
3. 关键数据可追溯：注明来源，财务交代口径。
4. 正文字号 ≥22px，并按内容块数量上调。
5. 颜色不超过 4 种，强调色只用于关键信息。

## 禁忌

| 禁忌 | 替代 |
|------|------|
| 无信息量的标题（"工作成果"） | 结论式标题 |
| 空洞定性描述 | 量化数据 + 图表 |
| 纯文字罗列结构信息 | 流程图/层级图等结构图 |
| 大型装饰性图片 | 数据图与结构图 |
| 无来源的数据引用 | 注明来源 |
| 颜色 >4 种、渐变/阴影/艺术字滥用 | 三色体系 + 扁平纯色，靠字重字号分层级 |

## 场景适配

- **绩效述职**：开篇个人介绍页（基本信息/岗位职责/任期）；自评用数据支撑，避免"勤奋负责"等空话；含"不足与改进"但不超过总量 15%；未来计划关联岗位目标并量化。
- **政务 / 国企报告**：以主题色或中国红、党旗红为主色、金色为强调；封面/章节可用红色块，内容页保持浅底确保可读；用"稳步推进""取得显著成效"等规范表述；成果须与上级年度目标逐项对标；政治符号仅在固定装饰位使用，整体庄重大气。
- **行业研报（事实部分）**：可接受极高密度，单页可含 2-3 张相关图表 + 简洁解读，数据表 ≥4 行；所有数据标注来源与坐标轴单位。
- **学术 / 科研报告与答辩**：按"背景与问题 -> 相关工作 -> 方法 -> 实验设计 -> 结果 -> 结论与展望"组织；语言客观严谨，严禁臆造，所有引用真实可追溯；正文引用标编号（[1]、[2-4]），末尾设独立参考文献页（中文默认 GB/T 7714，英文默认 APA）；公式用 LaTeX 渲染并附变量说明；图表用标准编号（图1/表2）配完整图注、坐标轴标签与单位。
