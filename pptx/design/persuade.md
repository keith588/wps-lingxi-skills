# 说服 · 提案

> **适用目的**：让受众改变看法、认同主张并采取行动——不只是知道，而是被打动、愿意投入。
> **典型场景**：项目提案、商业计划书、融资路演、创业大赛、投资建议、战略规划、竞品与市场的观点驱动分析、内部立项争取资源。
> **何时选它**：受众握有决策权或选择权，你要的是他们"点头、批钱、入局"，而非单纯了解情况。

## 设计理念

- **观点驱动**：每页一个核心论点，用数据与案例支撑；强调 "so what"——数据背后的含义与影响，而非罗列事实。
- **制造张力**：靠"现状 vs 可能"的对比推进叙事；先让受众感到不满或落差，再给出更好的未来，对比越强，说服力越强。
- **受众是主角**：你是引路的向导，不是英雄；通篇站在受众的利益与动机上说话，回答"这对我有什么好处"。
- **收益导向**：每个举措都明确关联量化产出，让决策者看清投入产出的因果。
- **高级克制**：专业稳重大气，自信但不浮夸；避免一切廉价或华丽元素。

## 叙事与结构

- 整体用"对比节奏"贯穿：在"现状/痛点（what is）"与"愿景/方案（what could be）"之间来回摆动，逐步推向高潮与行动号召。
- 常用框架二选一或融合：
  - **战略框架**：背景趋势 -> 问题与机遇 -> 战略目标 -> 执行路径 -> 资源需求 -> 预期收益与 ROI -> 风险与保障 -> 行动号召。
  - **融资框架**：问题 -> 解决方案 -> 产品 -> 市场规模 -> 商业模式 -> 竞争优势 -> 牵引力 -> 团队 -> 财务预测 -> 融资需求。
- 少而精：核心论证用一条主线串起，宁可页数少而每页有力（10/20/30 精神——每页一个核心信息、字够大、不堆文字）。
- 每页标题是论点而非章节名（✗"融资计划"；✓"本轮融资 5000 万，18 个月实现盈亏平衡"）。
- 预期收益必须量化，让决策者一眼看到投入产出比。

## 受众与信息密度

- 密度：中到高。每页聚焦一个论点，但允许更丰富的数据与图表充分论证。
- 决策者动机优先：他们最关心"风险、回报、可行性"——核心数据页（市场规模/财务预测/ROI）信息可密集，愿景页与总结页则留白以传递高级感。
- 降低决策阻力：用权威（团队背景、专家背书）与社会认同（客户 Logo、标杆案例、媒体报道）降低感知风险；用稀缺与时间窗（窗口期、限额）制造紧迫感；用一致性从小承诺引导至大承诺。所有手段必须真实，不可伪造或夸大。
- 每页 4-6 个关键信息点，核心论证页可达 7 个；超过即拆页，避免信息过载。
- 目标：10 分钟翻完后，决策者能清晰回忆起目标、执行路径与预期收益。

## 视觉与版式

- 基调：远见、权威、现代；传递战略高度与格局。
- CRAP 四原则：对比（用大小/颜色/字重制造差异，让重点跳出）· 重复（统一的视觉风格贯穿全篇）· 对齐（每个元素与其他元素有明确的视觉关联，禁止随意摆放）· 邻近（相关内容靠拢成组，无关内容用空白隔开）。
- 网格系统：用隐含栅格统一安排标题、正文、图表的层级、比例与对齐；根据内容灵活选用单栏/多栏/模块化/层级网格。网格是辅助而非束缚。
- 关键页（市场规模、牵引力、财务预测、ROI）可用深色背景 + 大数字强调，制造分量感。
- 常用布局：居中大数字（市场/ROI）、左右分栏（图表 + 解读 / 现状 vs 目标）、三四栏卡片（方案对比/竞争分析/阶段目标）、时间轴路线图（战略路径/里程碑）、目标-路径-收益三段式。
- 产品页、团队页用真实图片（产品截图、团队照片）增强可信度；图标辅助说明商业模式与战略框架。

## 配色与字体

- 配色稳重有力量，传递可信与权威；有品牌色则围绕品牌色构建。
- 推荐主色：低饱和深色调（深海蓝、海军蓝、深绿）+ 白/中性灰/米色；强调色一种，用于关键数字、增长指标、行动按钮，金/琥珀色可添高级感。
- 深色页：深底 + 白字 + 主题色强调；浅色页：浅底 + 深字 + 主色用于标题与强调。严禁高饱和荧光色、>3 种主色、对撞配色。
- 字体：标题无衬线加粗，有力现代；正文无衬线常规；大数字可用加粗或细体（超大字号下细体更显呼吸感与高级感）。
- 字号层级：封面标题 44-56px -> 页面标题 34-40px -> 正文 22-28px -> 脚注 12-16px；大数字展示 44-60px 配小字解释。
- 正文字号下限 22px，并随内容块数量上调：≤2 块 ≥32px；3 块 ≥28px；4 块 ≥26px；≥5 块 ≥22px。

## 数据与图表表达

- **引导视觉焦点**：视线天然跳向对比、一次只能追踪少量对象、且会自动编织因果叙事。据此用强调色只点要表达的那一项，隔离关键数据序列，把"感受"做进图表里。
- **图表要有观点**：图表是陈述型（传达明确结论）而非探索型（让人自己看数据）。先定本页核心观点，再将其有说服力地可视化；标题写结论而非图名，不只呈现数据，更点出"所以应该怎么做"。
- **先定信息，再选图表**：图表形式由"要表达的结论"决定，而非由数据类型决定（要比大小->柱状/条形；看趋势->折线；看占比->饼/环/堆叠；看相关->散点/气泡；看流程结构或系统->结构图；看分布->直方图）。
- **精简图表元素**：删除不增进理解的视觉噪声（多余网格线、无意义颜色、3D 效果、次坐标轴、冗余刻度），最大化数据墨水比，清除 chartjunk。
- **数据即说服力**：关键商业数据（市场规模、增长率、ROI、预期收益）突出且可信，所有结论有数据支撑、有来源。
- **市场规模**：TAM/SAM/SOM 用嵌套圆、漏斗或大数字 + 层级展示。
- **牵引力/增长**：折线图展示趋势并标注关键里程碑。
- **单位经济**：大数字 + 公式拆解（CAC、LTV、毛利率）。
- **竞争定位**：二维矩阵定位竞争格局，突出自身优势位。
- **ROI / 收益**：投入产出比做大数字，辅以成本分解与收益预测图。
- **风险**：概率×影响矩阵 + 缓解策略。
- **信任背书**：客户/合作 Logo 墙、团队背景一行（照片 + 姓名 + 行业经验）。

## 硬性承诺（不可妥协）

1. 每页标题是论点（结论），不是章节名。
2. 每页聚焦一个核心信息；超过 7 个要点即拆页。
3. 预期收益必须量化，明确投入产出关系。
4. 关键数据有来源、可信；竞争分析用具体指标而非空泛断言。
5. 正文字号 ≥22px 并按内容块数量上调；主色 ≤3 种。

## 禁忌

| 禁忌 | 替代 |
|------|------|
| 大段文字段落 | 提炼为要点与数据 |
| 无数据支撑的断言 | 附具体数字与来源 |
| 收益仅做定性描述 | 量化收益（ROI、增长率、节省成本） |
| 数据堆砌无结论 | 数据 + 洞察 + 行动建议三段式 |
| 信息过载（>7 要点/页） | 拆页，每页一个焦点 |
| 模糊的竞争分析 | 具体指标对比矩阵 |
|截断坐标轴/双标尺等误导 | 诚实呈现，靠观点而非把戏说服 |
| 通用图库照片、廉价渐变 | 真实产品/团队图，克制的高级视觉 |

## 场景适配

- **融资路演 / 创业大赛**：偏故事性，开场把痛点讲到让人有共鸣（现状 vs 可能的对比）；牵引力与单位经济是重头戏；融资需求清晰（金额、分配比例、关键里程碑）；deck 精炼，单页一个信息。
- **战略规划报告**：偏宏观格局，从行业趋势切入；强调对标与保障机制；每个举措的收益必须量化关联。
- **竞品与市场分析（观点驱动）**：避免纯罗列，每页以核心洞察作标题；"结论 -> 数据 -> 启示"三段式；观点鲜明、逻辑严谨、措辞专业但易懂。
- **内部立项**：聚焦"投入 vs 产出 vs 风险"，给决策者可直接拍板的信息密度。
- **公益倡导 / 主题宣传**：低碳环保、安全生产、健康促进等主题；用数据呈现现状与影响，用案例建立紧迫感，用明确的行动号召推动参与。
