# 教学 · 培训

> **适用目的**：让受众理解并掌握知识或技能——降低理解门槛、控制认知负荷，使学习者"看完能复述、能上手"。
> **典型场景**：中小学/高校课件、公开课与讲座、学生课堂演示、企业培训（入职/安全/技能）、教程与操作指南、学术内容的教学化讲解、文化介绍/科普。
> **何时选它**：受众是来"学会"的，成功标准是他们的理解与留存，而非被说服或获取一份报告。

## 设计理念

- **受众画像驱动一切**：先定受众是谁（年龄、知识水平、注意力特点），再据此定密度、配色、字号、语言风格。
- **管理认知负荷**：把与学习无关的"外在负荷"压到最低，让认知资源用于理解核心内容；信息量与受众认知容量匹配，既不超载也不稀疏。
- **循序渐进（分块）**：知识逐步展开，有铺垫、过渡、总结；复杂内容拆成可消化的小块。
- **视觉辅助理解**：图示、流程图、类比是降低门槛的工具而非装饰；用"图 + 讲解"的双通道呈现，避免文字与口述完全重复造成冗余。
- **每页一个知识点**：标题概括该页要点，读标题即知本页讲什么。

## 叙事与结构

- 单个知识点按探究式节奏展开：引入问题/现象（激发好奇、勾连已有经验）-> 探究/观察 -> 解释概念 -> 迁移应用到新情境 -> 小结/评价。
- 用提问激发思考（"为什么天空是蓝色的？"-> 下一页给解释），让学习者带着问题往下走。
- 用类比降低门槛（"电流像水流，电压像水压"），把抽象概念锚定到熟悉经验。
- 语言按受众调整（见下表）；面向学习者时用对话式、第二人称，比正式书面语更利于理解。

## 受众与信息密度

- 先判断受众面临的是知识缺口（缺信息）还是技能缺口（缺练习）——技能缺口需靠示范与练习弥补，仅给知识不够；新手先教基础再逐步加难，专家可直奔进阶。
- 把新内容挂在学习者已知的概念上以促进吸收；鼓励主动参与（提问、动手、讨论）提升留存。
- 压低与学习无关的外在认知负荷，让认知资源集中于核心内容；信息量与受众认知容量匹配，既不超载也不稀疏。
- 受众浮动密度：
密度随受众浮动，核心是"看完一页能用一句话概括这页讲了什么"：

| 受众 | 填充率 | 图文比例 | 语言风格 | 互动 |
|------|--------|---------|---------|------|
| 幼儿/小学 | 40-60% | 图主导(50%+) | 对话式、趣味、鼓励 | 大量(游戏化/问答) |
| 初中 | 60-70% | 均衡 | 引导式 + 适量术语 | 适中(思考题) |
| 高校/讲座 | 65-80% | 文字略多于图 | 术语 + 清晰解释 | 适中(讨论/案例) |
| 学生演示 | 60-70% | 均衡 | 自然、逻辑清晰 | 极少 |
| 企业培训 | 65-80% | 文字略多于图 | 实践导向、简洁 | 适中(动手/情景测验) |

- 每页 1-2 个知识点；超过 6 行的段落应分页或转为图示。
- 关键概念用大字/色块/图标强调，辅助信息用较小字号。
- 早教偏低密度、留白给消化空间；高校/企业培训可承载更丰富层次。

## 视觉与版式

- 基调：清晰、循序渐进、视觉辅助理解；亲和度按受众浮动（早教活泼、高校稳重）。
- CRAP 四原则：对比（用大小/颜色/字重制造差异，让重点跳出）· 重复（统一的视觉风格贯穿全篇）· 对齐（每个元素与其他元素有明确的视觉关联，禁止随意摆放）· 邻近（相关内容靠拢成组，无关内容用空白隔开）。
- 网格系统：用隐含栅格统一安排标题、正文、图表的层级、比例与对齐；根据内容灵活选用单栏/多栏/模块化/层级网格。网格是辅助而非束缚。
- 内容页以白色/浅色背景为主，深色仅用于封面/章节/强调页（投影上深底正文不可读）。
- 所有图示必须辅助理解，禁止纯装饰图（去杂：与学习无关的图、音、字一律删掉）。
- 常用布局：左图右文、上图下结、双栏对比、图示 + 要点、概念 + 示例；底部可放思考/练习卡片维持互动。

## 配色与字体

- 清晰可读第一，文字与背景足够对比，适配大教室投影。
- 幼儿/小学明亮活泼可多色；初中兼顾活力与专注；高校/讲座用稳定主色 + 一种强调色。
- 可用颜色编码知识类别（概念=蓝、案例=绿、注意=橙），语义色不超过 4 种。
- 字体：标题无衬线加粗，正文无衬线常规；公式用 LaTeX 渲染，代码用等宽字体。
- 字号层级（投影场景偏大）：封面 36-56px -> 页面标题 26-40px -> 正文 ≥22px -> 注释 12-18px；安全/警示信息加大加粗。早教取各档上限。
- 正文字号下限 22px，并随内容块数量上调：≤2 块 ≥32px；3 块 ≥28px；4 块 ≥26px；≥5 块 ≥22px。

## 数据与图表表达

- **信号化 (signaling)**：用箭头、高亮、序号引导视线，明确"先看哪、再看哪"，让结构一目了然。
- **分块 (segmenting)**：复杂流程拆成分步呈现，一次只推进一步，而非一页塞满全流程。
- **突出关键数据**：用大小、颜色、位置等前注意属性突出图表中的关键数据点，让结论不需要寻找。
- **先定信息，再选图表**：图表形式由"要表达的结论"决定，而非由数据类型决定（要比大小->柱状/条形；看趋势->折线；看占比->饼/环/堆叠；看相关->散点/气泡；看流程结构或系统->结构图；看分布->直方图）。
- **精简图表元素**：删除不增进理解的视觉噪声（多余网格线、无意义颜色、3D 效果、次坐标轴、冗余刻度），最大化数据墨水比，清除 chartjunk。
- **图示/流程图（核心）**：用形状（矩形/椭圆/圆角矩形）+ 箭头 + 文字构建概念关系图与步骤图，用真实形状而非图片占位；不同颜色区分输入/输出/核心概念。
- **最简化**：图表只保留服务于理解的元素，去掉多余刻度、网格、装饰，降低外在负荷。
- **对比表格**：概念对比、方法对比，差异用颜色或加粗突出。
- **对错对比**：左右两栏展示正确/错误做法，配绿/红指示，浅色卡片包裹（尤其适合规范说明与操作培训）。
- **编号步骤 / 核查清单**：操作流程用编号列表配图示，关键流程配清单供逐项核查。
- **关键词强调**：核心术语首次出现时加粗或用强调色。

## 硬性承诺（不可妥协）

1. 每页标题概括一个知识点，不是模糊章节名。
2. 所有图示服务于理解，禁止纯装饰图。
3. 每页知识点 ≤2 个；超 6 行正文须分页或转图示。
4. 内容页用浅底，确保投影可读；正文字号 ≥22px 并按内容块数量上调。
5. 文字与口述/旁白不完全重复（避免冗余效应）。

## 禁忌

| 禁忌 | 替代 |
|------|------|
| 与知识无关的装饰图 | 辅助理解的图示/插图 |
| 密集文字墙(>6 行正文) | 分页或替换为视觉解释 |
| 文字与口述完全重复 | 图 + 关键词，避免双通道冗余 |
| 大面积深色背景的内容页 | 浅底确保投影可读 |
| 字号过小(早教尤甚) | 适配教室投影的字号 |
| 纯文字罗列概念 | 配图示、表格、对比 |
| 企业培训中大段政策原文 | 提取要点 + 案例插图，全文放附录 |

## 场景适配

- **学生课堂演示**：核心是清晰呈现成果并说服老师同学；用"问题-方法-结果-结论"或"论点-证据-总结"结构；简洁克制、1 主色 + 1 次色；有时间限制，每页信息量适中。常见结构：封面（标题/姓名/日期）-> 目录 -> 背景/问题 -> 方法/过程 -> 结果/数据 -> 讨论/结论 -> 参考文献。
- **企业培训**：用企业品牌色，封面/章节体现企业标识；实践导向，编号列表 + 流程图展示步骤；信息分层（必知/应知/可知）；案例驱动替代抽象规定；对错对比 + 核查清单；安全/合规用高对比色（红/橙）+ 加粗 + 大字 + 祈使语气（"必须""严禁"），每条规则一句话。
- **学术内容教学化（讲座/课件）**：保留严谨性但降低门槛——先给直觉与类比，再给形式定义；公式配变量说明与图示；引用核心文献但不必像答辩那样列完整参考文献页（若需严格引用与参考文献页，见 inform.md 学术场景）。
- **早教/低龄**：超大字、强插画、游戏化互动；一页 1-2 个知识点，留足消化空间。
- **文化介绍 / 科普**：传统文化、历史人文、科学知识等主题；先用直觉和类比建立兴趣，再展开知识点；善用图片、时间线、对比图辅助理解；信息密度中等，兼顾视觉美感与内容充实。
