# 展示 · 推广

> **适用目的**：用视觉冲击与情感共鸣塑造印象、传递品牌/主题调性——让观众记住一个感觉、一个画面、一句话。
> **典型场景**：品牌发布、新品介绍、营销活动、艺术/时尚展示、主旨演讲（keynote）、高端形象展示。
> **何时选它**：成功标准是"被记住、被打动"，而非被说服下单或学会知识；观众为体验和感受而来。

## 设计理念

- **少即是多**：每页只传达一个核心信息，靠慷慨留白与极简排版营造高级感；克制是最高表达。
- **图像优势**：图像比文字更易被记住，且在曝光超过数秒后优势更明显——尽量用全屏出血大图替代文字堆砌。
- **留白是主动的**：留白不是"没填满"，而是有目的地引导视线、传递从容与高级；不要用 logo 或多余元素去填空白。
- **卖梦想，不卖参数**：靠颜色、图像、文案激发情感共鸣与向往，而非罗列功能规格；理性数据点到为止。
- **品牌/主题一致**：颜色、排版、基调高度统一，每页都是品牌身份的延伸。

## 叙事与结构

- 用三幕结构搭情感弧：铺垫（现状/期待）-> 冲突（落差/对手）-> 解决（亮相/高潮），先引入"对手"或问题，再让主角（产品/主题）登场。
- 一条主题线索贯穿全篇，有清晰情感节奏（铺垫 -> 高潮 -> 余韵）。
- 用"三的法则"组织要点：短期记忆一次只记得住三四件事，所以核心卖点/信息控制在三个，而非罗列二十个特性。
- 标语式标题：每页标题是一句精炼有力、可被记住的话（如"世界最薄笔记本"），而非功能描述。
- 长演示每隔一段切换呈现形式（大图、视频、实物演示、互动环节），用形式变化对抗注意力衰减。
- 精心设计一个"震撼时刻"：全篇至少有一页制造惊叹（极致视觉、戏剧化对比、出人意料的揭晓），让观众记住。

## 受众与信息密度

- 密度：低（填充率约 40-60%）。每页只一个核心信息，慷慨留白是刻意的设计选择而非内容不足。
- 观众为感受而来：关键数字、标语、核心概念用超大字体独占整页，赋予仪式感。
- 字越少，每个字分量越重；文案必须精雕细琢。
- 抽象大数用类比落地（"相当于 X 个""每天 Y 次"），让观众对数字有直觉体感。
- 目标：翻完后，观众脑海里留下的是几个画面和一句话，而非一串信息。

## 视觉与版式

- 基调：大胆、极简、情感驱动；每页都想让观众屏息。
- CRAP 四原则：对比（用大小/颜色/字重制造差异，让重点跳出）· 重复（统一的视觉风格贯穿全篇）· 对齐（每个元素与其他元素有明确的视觉关联，禁止随意摆放）· 邻近（相关内容靠拢成组，无关内容用空白隔开）。
- 网格系统：用隐含栅格统一安排标题、正文、图表的层级、比例与对齐；根据内容灵活选用单栏/多栏/模块化/层级网格。网格是辅助而非束缚。
- 鼓励打破常规网格：不对称构图（比对称更有张力）、出血图、超大排版、对角切割、圆形裁切。
- 用三分法布局：主体与文字落在三分线交点上，构图更稳更高级；非对称 + 留白激活画面。
- 常用布局：居中标语页、全出血图 + 半透明叠字、纯色/渐变背景 + 单行文案、左右不对称分割（比例可大胆偏斜如 70:30）、大数字独占页。
- 可无固定标题栏、无页码、无导航——每页只承载一条信息。

## 配色与字体

- 颜色即调性，是传递品牌标识与情感氛围的主要语言。
- 品牌发布：围绕品牌色 + 黑/白 + 一种强调色；时尚/艺术：高饱和对撞色、莫兰迪调或单色明暗渐变均可；文化推广：从主题意象取色（水墨灰、故宫红、敦煌金等）。
- 允许品牌/主题色系内的微妙渐变以增强氛围。
- 排版是设计感的核心载体：标题/标语用超大字号（40-68px），追求冲击力；艺术/文化场景可用衬线或书法字强化主题身份；正文用细体/常规保持优雅克制。
- 用字重对比（Bold vs Light）代替字号对比，层级更精致；字距行距精心调校，慷慨间距比紧密排列更显高级。
- 正文字号下限 22px；本场景内容块通常很少，正文取高档位（≤2 块 ≥32px）让字有分量。

## 数据与图表表达

- **大数字主导**：核心数据做成超大字独占一组，数字本身就是视觉主角，仅配一行极简注释。
- **引导视觉焦点**：用大小、颜色、位置等前注意属性把视线引向关键数据点，减少受众的认知负荷。
- **先定信息，再选图表**：图表形式由"要表达的结论"决定，而非由数据类型决定（要比大小->柱状/条形；看趋势->折线；看占比->饼/环/堆叠；看相关->散点/气泡；看流程结构或系统->结构图；看分布->直方图）。
- **极简图表**：若用图表，删到只剩数据本身——去掉网格线、无意义颜色、3D 效果、多余刻度（最大化"数据墨水"，清除 chartjunk）。
- **遮罩叠加**：半透明色块叠在图片上，保留质感同时确保文字可读。
- **出血图片**：图片延伸超出页面边缘，打破框架。
- **环境光/纹理**：径向渐变营造舞台级光影，纯色背景叠细微纹理（纸张/织物/金属）增加触感深度。

## 硬性承诺（不可妥协）

1. 每页只一个核心信息，慷慨留白（填充率 40-60%）。
2. 核心卖点/信息 ≤3 个（三的法则）。
3. 标题是标语，不是功能描述。
4. 优先用高质量大图替代文字堆砌；图表删到只剩数据。
5. 全篇视觉语言高度统一；正文字号 ≥22px 且通常取高档位。

## 禁忌

| 禁忌 | 替代 |
|------|------|
| 密集文字段落 | 提炼为短句或标语 |
| 复杂数据表格 | 大数字 + 极简图 |
| 低质量/带水印图片 | 高质量品牌素材或艺术摄影 |
| 通用剪贴画/图库人物 | 品牌定制视觉或高质量图标 |
| 单页信息过多(>3 卖点) | 拆页，每页一个焦点 |
| 企业模板感/通用外观 | 强化品牌或主题的独特视觉语言 |
| 图表带 3D/网格/装饰 | 去 chartjunk，只留数据 |
| 字号小、布局拥挤 | 大字号、宽间距、充足留白 |

## 场景适配

- **品牌发布 / keynote**：围绕品牌色与一条情感线；产品特写大图 + 极简规格；高潮页用全屏视觉 + 一句 slogan，对应精心设计的"震撼时刻"。
- **新品介绍**：真实产品图为主，避免虚构产品的生成图；特性用"大图 + 一行说明"逐条铺陈，卖点控制在三个。
- **艺术 / 时尚**：可用艺术摄影、展品图、书法/衬线字与诗意文案；配色从主题意象提取；避免说教，让视觉与文字让观众自己感受。
- **营销活动**：主视觉统一，行动号召（报名/扫码/参与）做成显眼的视觉焦点。
