# 通用

> **适用目的**：用途不明确，或同时夹杂告知/说服/教学/展示的混合场景时的默认兜底。
> **典型场景**：会议分享、项目介绍、工作沟通、内部讨论、综合性汇报。
> **何时选它**：判断不出单一主导目的时用它；一旦能判断，优先改用更贴切的专门预设。

## 选择更精准的预设（优先）

通用预设是兜底；若能判断主导目的，建议改用更贴切的预设以获得更好效果：
- 让受众获取事实/结论（汇报、总结、研报、述职、政务）-> inform.md
- 让受众认同并行动（提案、路演、战略、观点驱动分析）-> persuade.md
- 让受众理解掌握知识/技能（课件、培训、讲座、教程）-> teach.md
- 用视觉冲击塑造印象（发布、品牌、营销、艺术文化）-> showcase.md
- 介于多个目的之间时，选最主要的那个作为主预设，再借用其他预设的个别技巧。

## 设计理念

- **内容充实**：每页有实质内容，避免大面积留白或信息稀疏。
- **专业克制**：不过度装饰，也不过于朴素，保持适当美感与专业性。
- **结构清晰**：逻辑层级分明、信息有序，读者能快速抓取要点。
- **视觉丰富**：善用图表与结构图辅助传达，避免纯文字墙。

## 叙事与结构

- 每页标题概括该页核心要点，而非简单主题词，读标题即知主要信息。
- 信息按逻辑层级组织：一般 -> 具体，原因 -> 结果，问题 -> 解决方案。
- 语言简洁专业、数据驱动，避免模糊描述；关键结论与数据加粗，方便快速浏览。

## 受众与信息密度

- 密度：中高（填充率约 65-80%）。页面充实但不拥挤，每页承载一个完整主题并给出充分论据。
- 鼓励"图表 + 文字"在单页呈现多层次信息。
- 避免单张大图占满整页，或仅 2-3 条要点的稀疏页。
- 目标：读完一页感觉"信息完整、表达清晰"，既不空洞也不冗长。

## 视觉与版式

- 基调：专业、清晰、适度精致的通用商务风格。
- 用网格组织版面：以隐含的栅格安排标题、正文、图表，让元素对齐、比例协调、层级清晰；网格是辅助而非束缚，可按内容灵活取单栏/多栏/模块化布局。
- 应用 CRAP 四原则：对比（让重点跳出）、重复（统一视觉风格）、对齐（元素不随意摆放、彼此有视觉关联）、邻近（相关内容靠拢分组、无关内容用空白隔开）。
- 适度用结构图构建流程图、关系图、层级图；推荐布局：页面标题（顶部）-> 细灰色分隔线 -> 自由内容区 -> 页码（右下）。
- 以浅色调为主；科技等特殊主题可用适配的深色方案 + 白字。

## 配色与字体

- 用户提供品牌色时优先用品牌色。
- 三色体系：主色 + 灰阶 + 一种强调色，保持一致性；全文（含灰阶）不超过 4 种颜色，避免花哨多色。
- 字体：标题无衬线加粗，层级突出；正文无衬线常规，阅读舒适。
- 字号层级：封面标题 44-52px -> 页面标题 34-40px -> 副标题 28-32px -> 正文 22-28px -> 脚注 12-16px。
- 正文字号下限 22px，并随内容块数量上调：≤2 块 ≥32px；3 块 ≥28px；4 块 ≥26px；≥5 块 ≥22px。不要默认压到下限。

## 数据与图表表达

- **引导视觉焦点**：用大小、颜色、位置等前注意属性把视线引向关键数据点，减少受众的认知负荷。
- **按功能选图表**：先想清楚"要向受众传达什么"，再据此选型——比较差异->柱状/条形；趋势->折线；占比（部分对整体）->饼/环/堆叠；相关->散点/气泡；流程或系统运作->结构图；分布->直方图。
- **精简图表元素**：删除不增进理解的视觉噪声（多余网格线、无意义颜色、3D 效果、次坐标轴、冗余刻度），最大化数据墨水比，清除 chartjunk。
- **结构图**：流程图、层级图、循环图、关系图、矩阵表达逻辑关系，比纯文字直观。
- **表格**：多维对比、数据汇总，适合结构化信息。
- **KPI 大数字**：关键指标突出展示。
- **结构化列表**：每组要点 3-5 条，含具体数据。
- **强调标注**：关键数据与结论加粗。

## 硬性承诺（不可妥协）

1. 每页标题概括核心要点，不是主题词。
2. 内容充实，避免大面积留白或纯文字页。
3. 颜色不超过 4 种（三色体系）。
4. 元素对齐到隐含网格，遵循 CRAP 四原则。
5. 正文字号 ≥22px 并按内容块数量上调。

## 禁忌

| 禁忌 | 替代 |
|------|------|
| 过度装饰/华丽效果 | 简洁、专业的视觉设计 |
| 大面积留白 | 充实的内容填充 |
| 纯文字页面 | 图表或结构图配合的图文组合 |
| 颜色过多(>4 种) | 三色体系：主色 + 灰阶 + 强调色 |
| 装饰性图标过载 | 仅在辅助理解时使用图标 |
| 渐变和阴影滥用 | 扁平纯色为主 |
| 元素随意摆放 | 对齐网格，遵循 CRAP |
