# 体育学科 Rubrics 评价维度参考

体育课 Rubrics 评价维度的常见组合模式，供 Agent 在生成体育相关 Rubrics 时参考。

---

## 通用体育技能维度模板

适用于大多数体育技能课（球类、体操、武术、田径等）：

| 维度 | 关注点 | 适用场景 |
|------|-------|---------|
| 动作技能 | 动作规范性、连贯性、协调性 | 所有技能课 |
| 体能表现 | 速度、力量、耐力、灵敏等 | 田径、体能训练 |
| 运动参与 | 积极性、专注度、练习投入度 | 所有课型 |
| 合作与交流 | 团队配合、沟通能力、互助意识 | 小组活动、对抗赛 |
| 安全与纪律 | 安全意识、遵守规则、体育品德 | 所有课型 |
| 战术理解 | 战术意识、决策能力、应变能力 | 球类对抗、比赛 |
| 创新与表现 | 创编能力、表现力、审美意识 | 武术套路、体操、舞蹈 |

---

## 按课型推荐的维度组合

### 技能学习课（新授课）
- 动作技能（40%）
- 运动参与（20%）
- 安全与纪律（20%）
- 合作与交流（20%）

### 技能巩固课（复习课）
- 动作技能（30%）
- 体能表现（20%）
- 战术理解（20%）
- 合作与交流（20%）
- 安全与纪律（10%）

### 比赛/展示课
- 动作技能（25%）
- 战术理解（25%）
- 合作与交流（25%）
- 运动参与（15%）
- 创新与表现（10%）

### 体能训练课
- 体能表现（40%）
- 运动参与（25%）
- 动作技能（15%）
- 安全与纪律（20%）

### 理论课
- 知识理解（35%）
- 表达与交流（25%）
- 合作与探究（20%）
- 学习态度（20%）

## 按运动项目的特殊维度

| 运动项目 | 可补充的特殊维度 |
|---------|----------------|
| 武术 | 精神气质、攻防意识、套路表现力 |
| 篮球 | 位置感、攻防转换、团队战术执行 |
| 足球 | 跑位意识、传接球决策、攻防配合 |
| 体操 | 身体控制、节奏感、动作美感 |
| 田径 | 起跑/出发技术、途中节奏、冲刺能力 |
| 游泳 | 呼吸节奏、水感、换气技术 |
| 跳绳 | 节奏感、花样创新、协调性 |
