<a name="english"></a>
## English

A Claude Code skill that helps you write high-quality academic literature reviews. Provides systematic guidance from topic selection, literature search, evaluation, structure planning to final writing.

### Features

- **Systematic 5-Step Method** - Complete workflow from literature search to final writing
- **Critical Thinking** - Analysis, synthesis, and critical evaluation beyond simple summarization
- **Structured Organization** - Choose the best organizational structure based on your topic
- **Gap Identification** - Discover research gaps and propose future directions
- **Ready-to-use Templates** - Introduction, body, and conclusion templates included

### Installation

1. Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
mkdir -p ~/.claude/skills/literature-review
cp SKILL.md ~/.claude/skills/literature-review/
```

2. Restart Claude Code or start a new conversation

### Usage

Simply tell Claude your topic when you need to write a literature review:

```
Help me write a literature review about [your topic]
```

```
I need to write a literature review on [your topic], please guide me
```

### The 5-Step Method

| Step | Description |
|------|-------------|
| **1. Search Literature** | Define research question, create keyword list, search databases |
| **2. Evaluate & Select** | Assess relevance, credibility, impact, and timeliness |
| **3. Identify Themes** | Find trends, controversies, key papers, and research gaps |
| **4. Plan Structure** | Choose chronological, thematic, methodological, or theoretical order |
| **5. Write Paper** | Follow templates for introduction, body, and conclusion |

### Organizational Structures

- **Chronological** - Track topic development over time
- **Thematic** - Organize by recurring themes
- **Methodological** - Compare results from different research methods
- **Theoretical** - Discuss theories, models, and key concepts

### What's Included

- Keyword list examples
- Boolean operator search tips
- Literature evaluation criteria
- Writing templates for each section
- Complete paper structure template
- Common problems and solutions

<a name="chinese"></a>
## 中文

一个帮助你撰写高质量学术文献综述的 Claude Code 技能。提供从选题、文献检索、评估筛选、结构规划到最终写作的全流程系统性指导。

### 功能特点

- **系统性五步法** - 从文献搜索到最终写作的完整工作流程
- **批判性思维** - 不仅是总结，更注重分析、综合和批判性评估
- **结构化组织** - 根据主题特点选择最佳的组织结构
- **填补研究空白** - 发现研究空白并提出未来研究方向
- **即用模板** - 包含引言、正文、结论的写作模板

### 安装方法

1. 将 `SKILL.md` 文件复制到你的 Claude Code skills 目录：

```bash
mkdir -p ~/.claude/skills/literature-review
cp SKILL.md ~/.claude/skills/literature-review/
```

2. 重启 Claude Code 或开始新对话

### 使用方法

当你需要撰写文献综述时，直接告诉 Claude 你的主题：

```
帮我写一篇关于[你的主题]的文献综述
```

```
我要写[你的主题]方面的文献综述，请指导我
```

### 五步法撰写文献综述

| 步骤 | 说明 |
|------|------|
| **1. 搜索相关文献** | 确定研究问题、创建关键词列表、使用数据库搜索 |
| **2. 评估和选取文献** | 评估相关性、可信度、影响力、时效性 |
| **3. 确定主题和空白** | 识别趋势、争议、核心论文、研究空白 |
| **4. 规划组织结构** | 选择时间顺序、主题顺序、方法论顺序或理论顺序 |
| **5. 撰写论文** | 按照模板撰写引言、正文和结论 |

### 组织结构类型

- **按时间顺序** - 追踪主题随时间的发展
- **按主题顺序** - 按反复出现的主题组织
- **按方法论顺序** - 比较不同研究方法的结果
- **按理论顺序** - 讨论各种理论、模型和关键概念

### 包含内容

- 关键词列表示例
- 布尔运算符搜索技巧
- 文献评估标准
- 各部分写作模板
- 完整论文结构模板
- 常见问题与解决方案
