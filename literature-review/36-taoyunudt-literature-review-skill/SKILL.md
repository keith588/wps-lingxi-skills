# 文献综述写作 Skill

帮助用户撰写高质量的学术文献综述，提供系统性的方法论指导和写作框架。

## 核心理念

1. **系统性方法** — 遵循五步法，从文献搜索到最终写作
2. **批判性思维** — 不仅总结，更要分析、综合和批判性评估
3. **结构化组织** — 根据主题特点选择最佳的组织结构
4. **填补空白** — 最终目标是发现研究空白并提出未来方向

---

## 什么是文献综述

文献综述是指在某一时间段内，对某一领域、某一专业或某一方面的课题、问题或研究专题搜集大量相关数据、资料和主要观点，然后通过阅读、分析提炼、归纳整理当前课题、问题或研究专题的最新进展、学术见解或建议，对其做出综合性介绍和阐述的一种学术论文。

### 文献综述的目的

- 展示作者对研究和学术背景的熟悉程度
- 为作者本人的研究建立理论框架和方法
- 将作者本人与本领域其他研究人员和科研工作联系起来
- 寻找和展示研究如何填补学科空白或为学科发展做出贡献

## 五步法撰写文献综述

### 第一步：搜索相关文献

#### 1.1 确定研究问题

**独立文献综述**：必须选择一个重点，提出一个中心问题来指导文献搜索。问题必须能在不收集原始数据、没有开展实验的情况下，仅通过对现有文献的回顾来回答。

**研究论文中的文献综述**：研究问题一般是明确的，检索与研究问题相关的科研文献。

#### 1.2 创建关键词列表

- 列出每个关键概念或变量
- 列出所有同义词、近义词和相关术语
- 在搜索过程中发现新关键词时持续添加

**关键词示例**（以"地理空间数据库集成与更新方法"为例）：
```
核心概念：
- spatial database, Geodatabase, Multiple representation databases
- incremental update, incremental information

相关领域：
- Geographical Information System, Global positioning systems
- Land cover, Spatio-temporal data modelling
- spatial Data analytics, Coupling coordination evaluation
- Data matching, Cartographic generalization
```

#### 1.3 文献数据库

常用学术文献数据库：
- **综合类**：Web of Science, Google Scholar, SCI-Hub
- **理工科**：IEEE Xplore, ACM Digital Library, Elsevier ScienceDirect, Springer, IOP Science
- **医学/生命科学**：Medline, PubMed
- **人文社科**：JSTOR, Project Muse, EconLit
- **中文数据库**：中国知网(CNKI), 万方数据, 维普

#### 1.4 搜索技巧

**布尔运算符**：
- `AND`：查找包含多个关键字的来源
- `OR`：查找包含一系列同义词之一的来源
- `NOT`：排除包含某些术语的结果

**高效搜索策略**：
- 先阅读摘要，确定文献是否与研究问题相关
- 找到一篇契合的文献后，根据其参考文献寻找其他相关文献
- 特别注意被大量引用的文献
- 如果相同的作者、文献不断出现，一定要找到它们

---

### 第二步：评估和选取文献

#### 2.1 评估标准

对每篇文献问以下问题：

1. **相关性**：文献是否回答了我的研究问题？
2. **可信度**：信息来源是否可信？是否来自权威期刊？
3. **影响力**：论文被引用了多少次？高引用率意味着该文章具有影响力
4. **时效性**：
   - 科学领域：主要关注最新文献
   - 人文领域：可能需要从历史角度追溯概念演变

#### 2.2 文献管理

- 使用文献管理工具（Zotero, EndNote, Mendeley等）
- 阅读时做笔记，记录核心观点、数据
- 制作带注释的参考书目，包含完整引文信息和总结分析
- 加入自己的评论和理解，降低查重率

---

### 第三步：确定研究问题的主题、争议和不足

阅读文献时，需要梳理以下信息：

### 必须识别的五个要素

| 要素 | 说明 | 示例问题 |
|------|------|----------|
| **趋势和模式** | 理论、方法或结果随时间的变化 | 某些方法是否变得越来越流行或不流行？ |
| **研究主题** | 文献中反复出现的问题或概念 | 哪些概念在不同论文中都被讨论？ |
| **争议和矛盾** | 文献之间论点的区别或分歧 | 不同研究者对同一问题有何不同看法？ |
| **关键性核心论文** | 改变该领域方向的有影响力的理论或研究 | 哪些论文是该领域的里程碑？ |
| **空白和不足** | 文献中缺失的研究、需要解决的弱点 | 现有研究有哪些局限性？ |

#### 示例：研究空白识别

在回顾"不确定性感知可视化分析"文献时发现的不足：
```
Gap 1: The uncertain and conflicting patterns are not well-defined.

Gap 2: The problems we're facing are the data conflicts,
different resolution, missing and conflicting data, which
together lead to the uncertainty in the spatial temporal data.
```

### 第四步：规划和组织文献综述结构

四种组织结构可以单独使用或组合使用：

#### 4.1 按时间顺序 (Chronological)

**适用场景**：追踪主题随时间的发展

**注意事项**：
- 避免简单地按顺序列举文献
- 分析形成该领域方向的模式、转折点和关键论点
- 解释某些发展是如何以及为什么发生的

**示例结构**：
```
1. 早期阶段 (1990-2000)：理论的萌芽
2. 发展阶段 (2000-2010)：方法的多元化
3. 成熟阶段 (2010-2020)：应用的扩展
4. 当前趋势 (2020-至今)：与新技术的融合
```

#### 4.2 按主题顺序 (Thematic)

**适用场景**：发现研究问题存在反复出现的主题

**示例结构**（以"老龄化居民医疗水平"为例）：
```
1. 医疗政策影响因素
2. 文化层次影响因素
3. 经济收入影响因素
4. 医保基数影响因素
5. 家庭状况影响因素
```

#### 4.3 按方法论顺序 (Methodological)

**适用场景**：从不同研究方法或学科角度获取文献

**示例结构**：
```
1. 定量研究的主要发现
2. 定性研究的主要发现
3. 混合方法的研究结果
4. 不同方法论的比较分析
```

#### 4.4 按理论顺序 (Theoretical)

**适用场景**：讨论各种理论、模型和关键概念

**示例结构**：
```
1. 理论A及其应用
2. 理论B及其应用
3. 理论C及其应用
4. 理论整合与框架构建
```

### 第五步：撰写文献综述论文

#### 5.1 引言 (Introduction)

**必须包含**：
- 文献综述的研究背景
- 研究主题的重要性和意义
- 文献综述的范围和边界
- 研究问题或目标
- 文献综述的结构预告

**引言模板**：
```
[研究领域]在近年来得到了广泛关注，因为[重要性原因]。
然而，目前缺乏对[具体主题]的系统性综述。
本文旨在[研究目标]，具体回答以下问题：
[研究问题1]
[研究问题2]
本文首先[结构说明]，然后[结构说明]，最后[结构说明]。
```

#### 5.2 正文 (Body)

**写作原则**：

| 原则 | 说明 | 实践技巧 |
|------|------|----------|
| **总结和综合** | 概述每个文献的主要观点，结合成连贯的整体 | 不要逐篇罗列，要跨文献综合 |
| **分析和解释** | 添加自己的解释，讨论发现的意义 | 使用"这表明..."、"这意味着..." |
| **批判性评估** | 提及文献研究的优缺点 | 指出方法局限性、样本不足等 |
| **结构良好** | 使用过渡词和主题句 | "然而..."、"与之相反..."、"近年来..." |

**正文段落模板**：
```
[主题句：该小节的主要观点]
[文献支持1]的研究表明...（Author, Year）
[文献支持2]进一步发现...（Author, Year）
[综合分析]这些研究共同指出...
[批判性评价]然而，这些研究存在[局限性]...
[过渡到下一点]
```

#### 5.3 结论 (Conclusion)

**独立文献综述论文的结论应包含**：

1. **主要发现总结**：从文献中获得的核心结论
2. **研究空白识别**：当前领域内研究的差距
3. **未来研究方向**：对后续研究的具体建议
4. **理论/实践意义**：研究发现的学术和应用价值

**结论模板**：
```
本文对[研究主题]进行了系统性文献综述，得出以下主要结论：
1. [主要发现1]
2. [主要发现2]
3. [主要发现3]

通过文献分析，发现当前研究存在以下空白：
- [研究空白1]
- [研究空白2]

基于以上分析，未来研究可以从以下方向展开：
1. [未来方向1]
2. [未来方向2]
```

## 常见问题与解决方案

### Q1: 文献数量太多，如何筛选？

**解决方案**：
1. 设定明确的纳入/排除标准
2. 优先选择高影响因子期刊论文
3. 优先选择高被引论文
4. 关注近5年的最新研究
5. 从综述论文入手，追溯参考文献

### Q2: 如何避免只是简单罗列文献？

**解决方案**：
1. 使用综合而非罗列的写作方式
2. 找出文献之间的联系和对比
3. 加入自己的分析和评论
4. 使用"共同表明"、"与之相反"等连接词
5. 围绕主题而非文献来组织段落

### Q3: 如何发现研究空白？

**解决方案**：
1. 关注文献中"未来研究"部分的建议
2. 寻找不同研究之间的矛盾或不一致
3. 注意研究方法的局限性
4. 考虑新技术/新理论对现有研究的挑战
5. 思考研究结果的适用边界

## 使用此 Skill 的方法

当用户需要撰写文献综述时，按以下步骤引导：

1. **明确研究问题**：帮助用户明确文献综述的研究问题和范围
2. **制定检索策略**：协助用户确定关键词和数据库
3. **组织文献分析**：帮助用户识别主题、趋势、争议和空白
4. **规划结构**：根据内容特点建议最佳组织结构
5. **撰写指导**：提供具体的写作建议和模板
6. **修改完善**：帮助用户改进文献综述的质量
