# Chinese Sport-Science Breadth Resources

This layer supports routing across Chinese sport-science journals. Use it when
the user needs to choose between 体育科学, 体育与科学, 体育学刊, university sport
journals, or sport-technology outlets before writing for one target.

## What to Load

- `official-source-map.md` - official-source anchors and verification notes for
  each journal.
- `source-basis.md` - source policy, ranking basis, and cautious-use rules.
- `worked-examples/venue-routing.md` - worked routing cases for sport pedagogy,
  training science, sport sociology, sport management, and measurement papers.
- `exemplars/selection-patterns.md` - selection archetypes and sibling-journal
  traps for Chinese sport-science submissions.

## Usage Rule

Start with `cn-sport-journal-workflow` for uncertain targets. Then load only
the specific journal card that matches the manuscript's subject, method, and
institutional audience. Re-check current 投稿须知 before giving final formatting
or submission-channel advice.
