# 选刊原型（Venue-Selection Patterns）

以下为本广度合集的**路由原型**，非发表实例，也不替代各刊当前官方源信息。

## 声望不等于对口

体育学期刊常拒掉技术合格但"写给错误读者"的稿件。按稿件的**核心主张**路由：

- 学科一般性机制 / 方法学突破（运动人体科学或体育社科）：先试
  `china-sport-science`（《体育科学》）。
- 学理争鸣 / 跨界与思想性议题：试 `sports-and-science`（《体育与科学》）。
- 教学训练实践、区域与院校特色：试相应体育院校学报
  （北体 / 上体 / 成体 / 武体 / 沈体 / 天体 / 西体）。

## 自然科学方法 vs 人文社科方法

体育学横跨两类范式，期刊对方法的期待不同：

- **运动人体科学**（生理 / 生化 / 生物力学 / 运动医学 / 体质健康）：要求实验
  设计、样本量与统计、可重复；机制证据优先于相关。
- **体育人文社会学**（社会学 / 管理 / 经济 / 政策 / 史哲）：要求论证、文献定位
  与（实证类）识别策略或（质性类）资料与解释的严谨。
- 投稿前确认目标刊更看重哪一类，并据此组织方法与证据。

## 一般性 vs 院校 / 专项

- 面向全学科、问题与方法可推广 → 学会旗舰（《体育科学》/《体育学刊》）。
- 院校特色、专项实践、区域样本 → 体育院校学报更对口。
- 单队 / 单赛季 / 不可重复的描述性结果，先补可推广性与机制，否则属常见退稿。

## 投稿前务必回官网复核

字数与栏目、摘要与关键词、参考文献体例、伦理与受试者知情同意（涉人体实验）、
利益冲突与基金信息，均以各刊当前官方《投稿须知》为准。
