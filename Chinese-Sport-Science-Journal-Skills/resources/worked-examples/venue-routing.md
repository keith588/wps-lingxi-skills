# 选刊路由示例（Worked Routing Examples）

以下为帮助 router 在相近体育学期刊间做选择的**虚构诊断场景**，不得当作真实
投稿或发表事实呈现，也不替代各刊当前官方《投稿须知》。

## 场景 1：运动人体科学实验，机制清楚

稿件：一项设计良好的运动生理 / 生物力学 / 运动医学干预实验，有对照、样本量
说明与机制指标，结论对训练或健康有一般性意义。

路由：
- 先判断贡献是**学科一般性机制**还是某项目的应用细节。
- 偏一般机制、方法严谨、面向全学科读者 → 先试 `china-sport-science`
  （《体育科学》，学会旗舰）或 `china-sport-science-and-technology`。
- 偏院校教学训练实践 → 试相应体育院校学报（如
  `journal-of-beijing-sport-university` / `journal-of-shanghai-university-of-sport`）。

## 场景 2：体育人文社会学，问题驱动

稿件：以体育社会学 / 体育管理 / 体育政策 / 体育史哲为题，论证一个面向体育与
社会关系的"大问题"，方法为调查 / 质性 / 政策分析。

路由：
- 面向全学科、理论与现实意义突出 → `china-sport-science` 或
  `journal-of-physical-education`（《体育学刊》）。
- 偏思辨 / 学理争鸣与跨界对话 → `sports-and-science`（《体育与科学》）。
- 偏院校 / 区域实践 → 相应体育院校学报。

## 场景 3：竞技体育 / 运动训练应用

稿件：针对某专项的训练方法、技战术或体能监控，证据来自队伍或实验室。

路由：
- 方法学或监控指标有可推广的训练科学贡献 → `china-sport-science` /
  `china-sport-science-and-technology`。
- 专项训练实践与院校特色结合 → `journal-of-wuhan-sports-university` /
  `journal-of-chengdu-sport-university` 等院校学报。
- 缺一般性、仅单队单赛季无重复 → 多为退稿风险，先补可推广性再投。

## 场景 4：学校体育 / 民族传统体育

路由：
- 学校体育政策与课程改革、青少年体质健康 → `journal-of-physical-education` 或
  院校学报；面向全学科重大议题再试 `china-sport-science`。
- 民族传统体育与武术 → 院校学报中相关栏目，按地域与特色匹配。
