# 《天津体育学院学报》投稿（journal-of-tianjin-university-of-sport）

## 期刊定位

天津体育学院主办的 CSSCI 综合性体育学报，**对运动人体科学（运动生理生化、生物力学、运动医学、运动控制 / 神经科学）相对偏重**，同时也收运动训练与体育人文社会学。核心要求是**机制清楚 + 测量与实验严谨**：对自然科学路径的设计、对照、信效度与统计要求较高。偏经验总结、缺机制的稿件在此偏弱。

## 触发时机

- 题目涉及运动生理生化、生物力学、运动医学、运动控制 / 神经科学、运动与健康的机制研究
- 数据来自实验室 / 现场的生理生化测试、动作捕捉与建模、影像、可穿戴监测
- 稿件想走**机制导向、测量严谨**的运动人体科学路线
- 也接纳运动训练与体育人文社会学，但本刊的运动人体科学辨识度更高

## 选题偏好

1. **运动生理生化**：训练适应、代谢与机能、运动与慢病 / 健康的机制
2. **生物力学与运动控制**：动作机制、建模、神经肌肉控制
3. **运动医学与康复**：损伤机制与防护、运动康复
4. **运动训练与体育人文社会学**：训练规律；体育治理 / 社会问题（需规范）

## 方法

- 自然科学路径为辨识强项：随机 / 交叉设计、对照、测量信效度、统计功效、伦理审查与知情同意；把指标变化翻译成**生理 / 力学 / 训练学机制**
- 人文社科路径：问卷与量表、计量、政策 / 文本分析，需问题意识与理论对话
- 强调测试方案与数据**可复现**
- 描述性测量若揭示重要机制线索可接纳，但要讨论局限

## 结构与写作

- 引言立**机制问题**：关于何种生理 / 力学 / 控制机制，与哪条脉络对话
- 方法写到可被同行复现：受试者、仪器、流程、统计模型
- 结果与讨论强调**机制解释**与生理 / 训练学含义，而非仅显著性
- 文风学理严谨，避免经验总结腔

## 路由防误投

- 本刊"运动人体科学"辨识度高；机制导向、测量严谨的稿件优先本刊。
- 先比较：竞技实战与备战应用 → `china-sport-science-and-technology`；学科理论引领、综合分量最重 → `china-sport-science`；体育社会科学理论 → `journal-of-shanghai-university-of-sport` / `journal-of-sports-research`；体育史 / 民族传统体育 → `journal-of-chengdu-sport-university`；学校体育 → `journal-of-physical-education`。
- 常见误投：纯医学临床研究（非运动情境）回临床医学渠道。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从天津体育学院 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《天津体育学院学报》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 自然科学稿核对伦理审查与知情同意、数据 / 补充材料要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的机制问题，而非经验总结
- [ ] 受试者 / 对照 / 仪器 / 流程 / 统计可复现
- [ ] 指标变化翻译成生理 / 力学 / 训练学机制
- [ ] 伦理审查与知情同意齐备
- [ ] 讨论了局限与外推边界
- [ ] 文风严谨，无经验汇报腔

## 高频拒稿雷区

- 缺机制，仅报指标差异与显著性
- 设计无对照、样本与功效不足、方案不可复现
- 伦理 / 知情同意缺失
- 人文社科稿缺理论对话
- 把新仪器 / 新数据当贡献本身

## 决策口诀

- "运动生理 / 力学 / 控制机制 + 严谨测量" → 本刊对口
- "竞技实战与备战应用" → 《中国体育科技》
- "学科理论引领、综合分量足" → 《体育科学》
- "体育社会科学理论" → 《上海体育大学学报》/《体育学研究》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】运动生理生化 / 生物力学运动控制 / 运动医学康复 / 训练或社科
【机制问题】<一句话>
【测量严谨度】对照/信效度/功效/伦理充分 / 不足（需补）
【机制解释】生理/力学/训练学含义清晰 / 仅指标（需补）
【贡献】揭示/修正 <机制>
```
