# 《成都体育学院学报》投稿（journal-of-chengdu-sport-university）

## 期刊定位

成都体育学院主办的 CSSCI 综合性体育学报，**以体育史、民族传统体育、武术与养生、体育文化见长**，同时也收一般体育学稿件。成体在体育史学与中华传统体育领域有深厚传统，本刊因此对**史料 / 文化 / 民族传统体育**类稿件接纳度高、要求专业。核心要求是**史学 / 文化问题意识 + 材料扎实 + 学理规范**。

## 触发时机

- 题目涉及体育史（古代 / 近现代 / 制度史 / 思想史）、民族传统体育、武术、传统养生、体育文化与非遗
- 也接纳一般体育人文社会学与运动科学，但本刊特色栏目在史学 / 文化 / 民族传统体育
- 稿件依赖史料、田野或文化分析，需要懂行的评审
- 不确定该投以史学 / 文化见长的综合刊还是别的体育刊

## 选题偏好

1. **体育史**：古代与近现代体育、制度与思想、奥林匹克与国际体育史
2. **民族传统体育与武术**：技理、流派、传承、现代转化与国际传播
3. **传统养生与体医融合**：导引、中医与运动健康（人文 / 文化视角为主）
4. **体育文化与非遗**：身体文化、仪式、地方与族群体育

## 方法

- 史学路径：一手史料的考订与解读、制度与思想分析、历史比较；强调史料出处与史学规范
- 文化 / 民族志路径：田野调查、深度访谈、文化分析；强调材料厚度与反身性
- 一般社科 / 自然科学稿亦收，但需符合相应方法规范
- 描述与考据若有学术增量，本刊比纯实证刊更接纳

## 结构与写作

- 引言立**史学 / 文化问题意识**，说明与既有研究的对话与史料 / 材料来源
- 史学稿：史料引证规范、考订严谨、论从史出
- 文化 / 民族传统体育稿：从材料提炼概念与机制，与理论往返
- 文风学理，避免"罗列史实 / 复述故事"而无问题与解释

## 路由防误投

- 本刊在史学 / 文化 / 民族传统体育最具特色；这类稿件优先本刊。
- 先比较：一般综合分量 → `journal-of-beijing-sport-university` / `china-sport-science`；体育社会科学理论 → `journal-of-sports-research` / `journal-of-shanghai-university-of-sport`；思辨理论 → `sports-and-science`；应用 / 竞技 → `china-sport-science-and-technology`；运动人体科学实验 → `journal-of-tianjin-university-of-sport`。
- 常见误投：纯历史学 / 民俗学稿若与体育本体关系弱，应回史学 / 民族学渠道。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从成都体育学院 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《成都体育学院学报》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献（含史料引证）格式
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的史学 / 文化 / 民族传统体育问题意识
- [ ] 史料 / 材料来源清楚、引证规范、考订严谨
- [ ] 论从史出 / 从材料提炼概念，而非罗列复述
- [ ] 与既有研究有实质对话，学术增量明确
- [ ] 与体育本体的关系清楚（非纯史学 / 民俗学）
- [ ] 文风学理、格式规范

## 高频拒稿雷区

- 罗列史实 / 复述故事，无问题与解释
- 史料出处不清、考订草率
- 与体育本体关系弱，更像纯史学 / 民俗学
- 民族传统体育稿停在介绍，无理论提炼
- 格式与史料引证不规范

## 决策口诀

- "体育史 / 武术 / 民族传统体育 / 养生文化" → 本刊对口
- "一般综合分量" → 《北京体育大学学报》/《体育科学》
- "体育社会科学理论 / 思辨" → 《体育学研究》/《体育与科学》
- "实验 / 训练应用" → 《天津体育学院学报》/《中国体育科技》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】体育史 / 民族传统体育武术 / 养生体文化 / 一般社科
【问题意识】<史学/文化问题一句话>
【材料/史料】来源清楚·考订严谨 / 不足（需补）
【与体育本体关系】清楚 / 偏弱（需说明）
【贡献】推进 <史学/文化认识>
```
