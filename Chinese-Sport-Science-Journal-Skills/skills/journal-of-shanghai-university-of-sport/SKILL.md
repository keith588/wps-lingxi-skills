# 《上海体育大学学报》投稿（journal-of-shanghai-university-of-sport）

## 期刊定位

上海体育大学（原上海体育学院）主办的 CSSCI 体育学顶尖学报。综合性，但以**体育人文社会学、体育管理与政策**见长，国际化与理论对话意识较强，同时也刊运动科学。核心要求是**问题意识 + 理论对话 + 方法规范**，对体育社会科学类稿件的理论分量要求高。纯应用 / 训练经验稿、缺理论框架的"对策建议"在这里偏弱。

## 触发时机

- 题目涉及体育治理与政策、体育产业与经济、全民健身与公共体育服务、体育社会问题、体育全球化与国际比较
- 也接纳运动人体科学，但社科类稿件更需理论对话
- 稿件想走**理论性、国际化、规范**的体育社会科学路线
- 不确定该投社科偏强的综合刊还是专门的应用 / 训练刊

## 选题偏好

1. **体育治理与政策**：体育改革、公共体育服务、体育法治与制度
2. **体育产业与经济**：体育消费、产业组织、赛事与职业体育
3. **体育社会学与文化**：身体、健康、全民健身、体育与社会变迁
4. **运动科学**：运动人体科学与健康促进（需规范设计）

## 方法（社科偏重，自然科学并收）

- 社科路径：问卷与量表（信效度、结构方程）、计量、政策与文本分析、比较研究、访谈 / 案例，强调与体育社会科学理论的实质对话
- 自然科学路径：实验 / 测试 / 干预，交代受试者、对照、信效度、统计与伦理
- 鼓励规范的混合方法与国际比较
- 描述性事实若揭示重要结构，需提升到理论解释

## 结构与写作

- 引言立**理论问题**，进入体育社会科学（治理、产业、社会学）脉络
- 把系数 / 材料翻译成**体育治理 / 社会过程**，不停在"对策三条"
- 重视概念清晰、论证层次与国际可对话性
- 文风学理、克制，避免政策汇报腔与纯技术腔

## 路由防误投

- 本刊是"社科偏强的综合学报"通道；社科稿的理论分量是关键门槛。
- 先比较：学科理论引领、综合分量最重 → `china-sport-science`；应用 / 竞技训练 → `china-sport-science-and-technology`；体育社会科学理论 / 思辨 → `journal-of-sports-research` / `sports-and-science`；体育史 / 民族传统体育 → `journal-of-chengdu-sport-university`；学校体育 → `journal-of-physical-education`。
- 常见误投：体育只是案例的经济 / 管理实证应回经管刊（见 `cn-sport-journal-workflow`）。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从上海体育大学 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《上海体育大学学报》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 社科稿核对是否需提供调查工具 / 访谈提纲；自然科学稿核对伦理与数据要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的理论问题，而非"对策建议"
- [ ] 与体育社会科学理论脉络实质对话
- [ ] 方法规范（量表信效度 / 计量识别 / 文本分析透明）
- [ ] 结果翻译成体育治理 / 社会过程
- [ ] 论证层次清晰、概念准确、可国际对话
- [ ] 无政策汇报腔与纯技术堆砌

## 高频拒稿雷区

- 缺理论框架的对策报告
- 社科稿把显著性当结论，无社会过程解释
- 概念含混、论证跳跃
- 仅本地经验、缺乏更一般的理论意义
- 自然科学稿设计与伦理不规范

## 决策口诀

- "体育治理 / 产业 / 社会 + 理论对话" → 本刊对口
- "纯应用 / 训练实战" → 《中国体育科技》
- "思辨 / 方法论反思的理论文章" → 《体育与科学》
- "规范社会科学问题意识" → 《体育学研究》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】体育治理政策 / 体育产业经济 / 体育社会学 / 运动科学
【理论问题】<一句话>
【理论对话】<治理/产业/社会学理论> | 缺失
【机制/解释】社会过程清晰 / 仅对策（需补）
【贡献】推进/修正 <理论>
```
