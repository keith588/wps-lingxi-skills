# 《体育与科学》投稿（sports-and-science）

## 期刊定位

江苏省体育科学研究所主办的 CSSCI 体育学刊物，以**思辨性、跨学科、理论批判与方法论反思**著称，常刊学术对话 / 深度访谈、问题与方法的论辩、概念与理论的批判性文章。核心要求是**原创的理论 / 方法论贡献 + 思想density + 论证力**：它推动"学科怎么想问题"，而非又一篇标准实证或对策报告。常规问卷实证、应用训练稿在此偏弱。

## 触发时机

- 题目是体育学的概念 / 理论 / 方法论问题，或对某研究范式 / 议题的批判性反思
- 文章形态偏理论论辩、学术对话、跨学科思想性写作
- 有原创概念框架或方法论主张，想与学界对话
- 不确定该投思辨理论向（本刊）还是规范实证社科向（《体育学研究》/《上海体育大学学报》）

## 选题偏好

1. **理论与概念**：身体、运动、健康、体育的概念重构与理论建构
2. **方法论反思**：体育研究的范式、方法与认识论批判
3. **跨学科对话**：体育与哲学、社会理论、文化研究、科学技术论的交叉
4. **学术对话 / 访谈**：与重要学者 / 议题的深度对话（本刊特色体裁）

## 方法（理论 / 思辨为主）

- 以**概念分析、理论建构、批判论证、跨学科综述**为主；要求论证严密、思想原创
- 经验材料可用，但服务于理论 / 方法论命题，而非作为主贡献
- 学术对话 / 访谈需有清晰的问题线索与思想增量
- 不以"识别干净"或"显著性"为价值标准

## 结构与写作

- 立**理论 / 方法论问题**：要重构什么概念、批判什么范式、提出什么主张
- 论证讲层次与思想density，与经典与当代理论往返
- 跨学科写作要消化而非拼贴，形成自己的命题
- 文风思辨、克制而有锋芒，避免空泛宏大叙事与术语堆砌

## 路由防误投

- 本刊是"思辨 / 理论 / 方法论"通道；原创理论与论证力是关键门槛。
- 先比较：规范实证体育社会科学 → `journal-of-sports-research` / `journal-of-shanghai-university-of-sport`；体育史 / 民族传统体育 → `journal-of-chengdu-sport-university`；学校体育 → `journal-of-physical-education`；运动人体科学实验 → `journal-of-tianjin-university-of-sport`；学科理论引领、综合分量最重 → `china-sport-science`。
- 常见误投：标准问卷实证、应用训练总结、对策报告，应投相应实证 / 应用刊。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从江苏省体育科学研究所 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《体育与科学》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 学术对话 / 访谈类体裁核对其特定要求与授权
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有原创的理论 / 方法论问题与主张
- [ ] 论证严密、思想有density，与理论往返
- [ ] 跨学科是消化整合，而非术语拼贴
- [ ] 经验材料服务于理论命题，而非充当主贡献
- [ ] 与学界有真对话，思想增量明确
- [ ] 无空泛宏大叙事与术语堆砌

## 高频拒稿雷区

- 标准问卷实证 / 应用训练总结投错门
- 宏大叙事 / 口号化，缺严密论证
- 跨学科拼贴术语，无自己的命题
- 文献综述堆砌，无批判与建构
- 对策报告冒充理论文章

## 决策口诀

- "概念重构 / 方法论批判 / 跨学科思想" → 本刊对口
- "规范实证体育社会科学" → 《体育学研究》/《上海体育大学学报》
- "运动人体科学实验 / 应用训练" → 《天津体育学院学报》/《中国体育科技》
- "体育史 / 民族传统体育" → 《成都体育学院学报》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】理论概念 / 方法论反思 / 跨学科对话 / 学术访谈
【理论问题】<要重构/批判/主张什么一句话>
【原创性】概念/方法论有增量 / 偏综述或对策（需补）
【论证力】严密·有density / 宏大空泛（需补）
【贡献】推进 <学科怎么想问题>
```
