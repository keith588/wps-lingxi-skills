# 《沈阳体育学院学报》投稿（journal-of-shenyang-sport-university）

## 期刊定位

沈阳体育学院主办的 CSSCI 综合性体育学报。地处东北，**冰雪运动（冬季项目）研究是其可识别的特色之一**，同时全谱覆盖运动训练与竞技体育、运动人体科学、体育人文社会学、学校体育。核心要求是**学科本体真问题 + 规范设计 + 扎实证据**；冰雪 / 冬季项目相关稿件在此有相对自然的归属。

## 触发时机

- 题目涉及冰雪运动、冬季项目训练 / 备战 / 普及（尤其后冬奥背景）
- 或落在体育学其他子学科，研究设计较完整
- 想找一本综合性体育学 CSSCI 刊，且稿件与冰雪 / 东北体育有关时尤宜
- 不确定该投综合学报还是专门刊

## 选题偏好

1. **冰雪与冬季项目**：训练与备战、技战术、群众性冰雪、冰雪产业与政策
2. **运动训练与竞技体育**：训练方法与规律、技战术、专项体能、后备人才
3. **运动人体科学**：运动生理生化、生物力学、运动医学与健康促进
4. **体育人文社会学与学校体育**：体育治理、全民健身、体育教学、青少年体质

## 方法（自然科学 / 人文社科并重）

- 自然科学路径：实验 / 测试 / 干预，交代受试者、对照、信效度、统计功效与伦理
- 人文社科路径：问卷与量表、计量、政策 / 文本分析、访谈 / 田野，强调问题意识与理论对话
- 鼓励规范的混合方法
- 看重问题—设计—证据—结论的闭环完整性

## 结构与写作

- 引言立**学科本体问题**与文献定位，说明边际贡献（冰雪稿可点明项目与情境特殊性）
- 自然科学稿：方法可复现，指标翻译成运动学 / 健康含义
- 人文社科稿：进入体育社会科学脉络，结果翻译成体育治理 / 社会过程
- 文风学理规范，避免训练经验汇报腔与纯技术堆砌

## 路由防误投

- 本刊综合且以冰雪 / 冬季项目见识；冰雪相关稿件优先考虑本刊。
- 先比较：体育史 / 民族传统体育 → `journal-of-chengdu-sport-university`；强理论体育社会科学 → `journal-of-shanghai-university-of-sport` / `journal-of-sports-research`；运动人体科学实验偏重 → `journal-of-tianjin-university-of-sport`；应用 / 竞技 → `china-sport-science-and-technology`；学科理论引领 → `china-sport-science`；学校体育 → `journal-of-physical-education`。
- 常见误投：体育只是案例的经管 / 临床稿，回对应学科渠道（见 `cn-sport-journal-workflow`）。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从沈阳体育学院 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《沈阳体育学院学报》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 自然科学稿核对伦理审查与知情同意、数据 / 补充材料要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的体育学本体问题与文献定位
- [ ] 研究设计完整、证据扎实
- [ ] 自然科学：受试者 / 对照 / 信效度 / 统计 / 伦理齐备
- [ ] 人文社科：理论对话与概念提炼到位
- [ ] 冰雪稿点明项目 / 情境特殊性与一般意义
- [ ] 写作规范，无汇报腔与炫技

## 高频拒稿雷区

- 选题平庸、边际贡献不清
- 设计粗糙、样本与功效不足、方案不可复现
- 人文社科稿停在描述或政策复述
- 冰雪稿停在工作总结，无研究问题
- 文献与格式不规范

## 决策口诀

- "冰雪 / 冬季项目" → 本刊较自然
- "体育学任一子学科 + 完整设计" → 本刊或其他综合学报
- "体育史 / 民族传统体育" → 《成都体育学院学报》
- "运动人体科学实验" → 《天津体育学院学报》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】冰雪冬季项目 / 运动训练竞技 / 运动人体科学 / 体育人文社会学
【问题意识】<体育学本体问题一句话>
【证据完整性】问题—设计—证据—结论闭环 / 有断点（需补）
【贡献】推进 <子学科认识>
【更优去向?】若有强专门归属，建议改投的体育刊
```
