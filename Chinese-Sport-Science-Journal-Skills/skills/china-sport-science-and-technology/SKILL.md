# 《中国体育科技》投稿（china-sport-science-and-technology）

## 期刊定位

国家体育总局体育科学研究所主办的 CSSCI 体育学来源刊，与《体育科学》同出体科所 / 学会体系，但分工不同：《体育科学》偏**学科理论与综合分量**，《中国体育科技》偏**应用、竞技体育与运动训练实战**。核心要求是**面向训练 / 备战 / 运动表现的真问题 + 可落地的科学证据**：测试与监控严谨、结论能指导实践。纯理论思辨或与竞技 / 健康实践脱节的稿件在这里会显得"用不上"。

## 触发时机

- 题目涉及竞技体育训练、运动选材、体能训练、技战术分析、负荷与疲劳监控、运动生理生化监测、反兴奋剂、运动康复与损伤防护
- 数据来自运动队 / 实验室的测试、监控与干预（运动表现、生理生化指标、动作 / 影像分析、可穿戴监测）
- 文章有明确的"指导训练 / 备战 / 健康促进"的应用落点
- 不确定该投综合理论向（《体育科学》）还是应用竞技向（本刊）

## 选题偏好

1. **竞技体育与运动训练**：训练方法与效果、周期与负荷、专项体能、技战术诊断、青少年选材与培养
2. **运动人体科学应用**：训练监控的生理生化指标、运动表现与恢复、运动营养与机能状态
3. **运动医学与康复**：损伤流行病学与防护、运动康复方案、运动与健康促进的干预
4. **体育科技应用**：可穿戴 / 影像 / 数据分析方法在训练与测试中的应用

## 方法

- 以**实验 / 准实验 / 干预 / 监控**为主：明确受试者与运动水平、对照与设计、测试方案与仪器、统计功效
- 强调测量**信效度**与方案**可复现**，把指标变化翻译成**训练学 / 运动表现含义**
- 接纳现场（运动队）研究，但要交代取样、混杂与局限
- 涉及人体试验须有伦理审查与知情同意；涉及兴奋剂 / 药物须合规

## 结构与写作

- 引言落在**训练 / 备战 / 健康的实际问题**，而非泛泛的理论缺口
- 方法部分把测试与干预写到"可被同行复现"的程度
- 结果与讨论强调**对实践的指导意义**：怎么用、对谁有效、边界条件
- 文风务实、数据扎实，避免空泛的理论包装与训练经验汇报腔

## 路由防误投

- 本刊定位"应用 / 竞技通道"；只有当核心贡献能**指导训练、备战、运动表现或健康实践**时才最对口。
- 先比较：理论与综合分量更足、面向学科本体 → `china-sport-science`（《体育科学》）；体育社会科学问题 → `journal-of-sports-research` / `journal-of-shanghai-university-of-sport`；运动人体科学基础研究亦可看 `journal-of-tianjin-university-of-sport`。
- 常见误投：纯医学临床研究（非运动情境）回临床医学渠道；体育只是案例的经管实证回经管刊（见 `cn-sport-journal-workflow`）。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从官方来源锚点（体科所 / 期刊官网 / 官方采编系统 / 官方 CNKI 页面）开始核验，并说明核验日期。
- 检索"《中国体育科技》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 核对伦理审查与知情同意、数据 / 补充材料要求，以及反兴奋剂 / 合规相关声明
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的训练 / 备战 / 健康应用落点，而非纯理论
- [ ] 受试者运动水平、对照、测试方案、统计功效交代完整
- [ ] 指标变化被翻译成训练学 / 运动表现含义
- [ ] 结论给出可操作的实践指导与边界条件
- [ ] 伦理 / 知情同意 / 合规齐备
- [ ] 无空泛理论包装、无训练经验汇报腔

## 高频拒稿雷区

- 与训练 / 竞技 / 健康实践脱节，"用不上"
- 监控 / 干预无对照、样本与功效不足、方案不可复现
- 把显著性当结论，无运动表现解释
- 现场研究不交代取样与混杂，外推过度
- 把新仪器 / 新数据当贡献本身

## 决策口诀

- "指导训练 / 备战 / 运动表现 + 严谨监控" → 本刊对口
- "学科理论 / 综合分量足" → 《体育科学》
- "体育社会科学问题意识" → 《体育学研究》/《上海体育大学学报》
- "运动人体科学基础机制" → 《天津体育学院学报》/《体育科学》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】竞技训练 / 运动人体科学应用 / 运动医学康复 / 体育科技应用
【应用落点】<指导何种训练/备战/健康实践一句话>
【证据强度】设计/对照/功效充分 / 不足（需补）
【实践指导】可操作 + 边界条件清晰 / 仅报指标（需补）
【贡献】改进 <训练/监控/康复实践>
```
