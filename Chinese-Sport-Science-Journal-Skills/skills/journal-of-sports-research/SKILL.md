# 《体育学研究》投稿（journal-of-sports-research）

## 期刊定位

南京体育学院主办的 CSSCI 体育学刊物，**聚焦体育社会科学 / 体育人文社会学**（体育社会学、治理、政策、史哲）。核心要求是**规范的社会科学问题意识 + 实质理论对话 + 方法严谨**。它偏学理与规范，接纳定量与定性，但共同门槛是"与体育社会科学理论的实质对话"，而非对策汇报或纯描述。

## 触发时机

- 题目涉及体育社会学、体育治理与政策、体育史与体育哲学、身体与社会、体育与国家 / 文化
- 数据是调查问卷、计量、政策 / 文本、访谈 / 田野、历史比较材料
- 文章想走**规范、理论性**的体育社会科学路线
- 不确定该投规范社会科学向（本刊）还是思辨向（《体育与科学》）或社科偏强综合刊（《上海体育大学学报》）

## 选题偏好

1. **体育社会学**：身体、健康、阶层与流动、性别、社会变迁中的体育
2. **体育治理与政策**：体育改革、公共体育服务、制度与法治
3. **体育史与体育哲学**：思想史、制度史、体育哲学与伦理
4. **体育与国家 / 文化**：体育外交、民族认同、体育文化

## 方法（定量 / 定性并重，重理论）

- 定量：问卷与量表（信效度、结构方程）、计量、文本量化；强调**机制与解释**而非识别军备竞赛
- 定性：深度访谈、田野民族志、历史比较；强调材料厚度、概念提炼与可信度
- 混合方法可加分，每种方法都要规范
- 描述性事实若揭示重要社会结构，需提升到理论解释

## 结构与写作

- 引言立**体育社会科学问题意识**，进入社会学 / 政治学 / 史学脉络
- 把系数 / 材料翻译成**社会过程 / 治理机制 / 历史脉络**
- 重视概念清晰、论证层次与理论往返
- 文风学理、克制，避免政策汇报腔与纯技术腔

## 路由防误投

- 本刊是"规范体育社会科学"通道；理论对话是关键门槛。
- 先比较：社科偏强但更综合 / 国际化 → `journal-of-shanghai-university-of-sport`；思辨 / 方法论反思的理论文章 → `sports-and-science`；体育史 / 民族传统体育专门向 → `journal-of-chengdu-sport-university`；学校体育 → `journal-of-physical-education`；学科理论引领、综合分量最重 → `china-sport-science`。
- 常见误投：体育只是案例的经济 / 管理 / 社会学实证应回对应学科渠道（见 `cn-sport-journal-workflow`）。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从南京体育学院 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《体育学研究》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 核对是否需提供调查工具 / 访谈提纲；史学稿核对史料引证规范
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有规范的体育社会科学问题意识，而非对策建议
- [ ] 与社会学 / 政治学 / 史学理论实质对话
- [ ] 定量：机制 / 异质性有社会学解释；定性：材料厚、编码透明、概念提炼到位
- [ ] 结果翻译成社会过程 / 治理机制 / 历史脉络
- [ ] 结论回扣并推进理论
- [ ] 无政策汇报腔与纯技术炫技

## 高频拒稿雷区

- 缺理论框架的对策报告
- 定量稿把显著性当结论，无社会过程解释
- 定性稿停在故事复述，无概念提炼
- 概念含混、论证跳跃
- 把方法 / 数据当贡献本身

## 决策口诀

- "体育社会科学 + 规范 + 理论对话" → 本刊对口
- "思辨 / 方法论反思的理论文章" → 《体育与科学》
- "社科偏强 + 国际化综合" → 《上海体育大学学报》
- "体育史 / 民族传统体育专门" → 《成都体育学院学报》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】体育社会学 / 治理政策 / 史哲 / 体育与国家文化
【问题意识】<体育社会科学问题一句话>
【理论对话】<社会学/政治学/史学理论> | 缺失
【机制/解释】社会过程清晰 / 仅系数或对策（需补）
【贡献】推进/修正 <理论>
```
