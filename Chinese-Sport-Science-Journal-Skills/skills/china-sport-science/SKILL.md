# 《体育科学》投稿（china-sport-science）

## 期刊定位

中国体育科学学会主办，国内**体育学第一刊**（CSSCI、中文核心、AMI 权威、国家社科基金资助期刊）。它是一本**综合性体育科学**期刊，横跨**运动人体科学**（运动生理生化、生物力学、运动医学、体质与健康）与**体育人文社会学**（体育社会学、体育管理、体育经济、体育政策、体育史哲、学校体育、运动训练与竞技体育、民族传统体育）。核心要求是**真问题 + 学科规范的证据**：自然科学路径要实验/测试严谨、机制清楚；人文社科路径要问题意识、理论对话与方法规范。把别的学科（纯经济学政策评估、纯教育学经验总结）的稿子原样套进来，会显得"与体育学本体无关"。

## 触发时机

- 题目落在运动训练、竞技表现、运动生理生化、生物力学、运动医学、体质健康、体育社会学、体育管理/经济/政策、学校体育、民族传统体育
- 数据是实验室/现场测试（生理生化指标、运动表现、动作捕捉、影像）、大型体质/健康监测、体育调查问卷、政策文本或田野材料
- 文章像"别学科的实证套了个体育外壳"，缺体育学本体问题
- 不确定运动人体科学稿与体育人文社会学稿各自怎么写到位

## 选题偏好

1. **运动人体科学**：训练适应与运动表现的生理生化机制、生物力学建模、运动损伤与防治、运动与慢病/体质健康
2. **竞技体育与运动训练**：训练理论与方法、技战术分析、负荷监控、青少年选材与长期发展
3. **体育人文社会学**：体育治理与政策、体育产业与经济、全民健身与公共体育服务、体育社会问题、体育史与体育哲学
4. **学校体育与体质健康**：体育教育改革、青少年体质与健康促进、运动干预

## 方法（自然科学 / 人文社科并重）

- **自然科学路径**：随机对照/交叉设计的运动干预、生理生化测试、生物力学测量与建模、运动表现与负荷监控；强调样本与对照、测量信效度、统计功效、可重复的测试方案
- **人文社科路径**：问卷与量表（信效度、结构方程）、计量分析、政策与文本分析、深度访谈/田野、历史比较；强调问题意识与理论对话，而非堆变量
- 跨学科混合方法可加分，但每条路径都要做规范
- 描述性的体质/参与监测若揭示重要结构性事实，本刊比纯方法刊更接纳

## 结构与写作

- 引言立**体育学本体问题**：这是关于何种运动机制 / 训练规律 / 体育制度的问题，与哪条学科脉络对话
- 自然科学稿：受试者与伦理（知情同意、伦理审查）、测试与干预方案、指标与仪器、统计分析交代清楚；把指标变化翻译成**运动生理 / 训练学含义**
- 人文社科稿：进入体育社会科学的理论脉络，把系数/材料翻译成**体育治理 / 社会过程**，不停在"X 对 Y 显著"
- 文风学理、节制，避免训练经验汇报腔与纯技术堆砌

## 路由防误投

- 本刊快速定位属于"综合体育科学"通道；只有当稿件核心贡献是**体育学本体问题**（运动机制、训练规律、体育制度/社会过程）时，才把《体育科学》作为首选。
- 同学科相近刊先比较：《中国体育科技》（更偏应用与竞技实战）、《北京体育大学学报》《上海体育大学学报》《武汉体育学院学报》《体育学刊》《体育与科学》《体育学研究》等体育学 CSSCI 来源刊。若这些刊的读者对象更贴近，不要因为名气强行投本刊。
- 常见误投：纯经济学/管理学实证只是"以体育为案例"，应回到对应经管刊（见 `cn-journal-workflow`）；纯医学临床研究应回到临床医学渠道；纯教育学经验总结缺体育学机制的，不对口。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索"《体育科学》 投稿须知 / 征稿简则 / 作者指南"，以期刊官网（中国体育科学学会主办）、官方采编系统或期刊官方 CNKI 页面为准；不要沿用二手代投网站格式
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/中图分类号、参考文献格式
- 自然科学稿核对伦理审查与知情同意、试验注册（如适用）、数据与补充材料要求；人文社科稿核对是否需提供调查工具/访谈提纲
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的体育学本体问题，而非"把体育当案例"
- [ ] 自然科学：受试者/伦理/测试方案/统计交代完整，机制解释到位
- [ ] 人文社科：与体育社会科学理论脉络有实质对话
- [ ] 指标/系数被翻译成运动学或体育治理含义，不止报显著性
- [ ] 结论回扣并推进了体育科学的认识
- [ ] 无训练经验汇报腔、无纯方法炫技

## 高频拒稿雷区

- 别学科实证套体育外壳，体育学本体问题缺位
- 运动干预无对照/无伦理/样本与功效不足，测试方案不可复现
- 问卷稿信效度与结构不交代，把显著性当结论
- 人文社科稿停在故事或政策复述，无理论提炼
- 把新仪器/新数据/新模型当贡献本身

## 决策口诀

- "运动机制 / 训练规律 + 严谨测试" → 本刊运动人体科学通道对口
- "体育制度 / 治理 / 社会过程 + 理论对话" → 本刊体育人文社会学通道对口
- "更偏竞技实战与应用" → 可看《中国体育科技》
- "落点是经济因果 / 管理构念，体育只是场景" → 回到经管刊（用 `cn-journal-workflow` 路由）

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】运动人体科学(实验/测试) / 体育人文社会学(定量/定性/政策) / 跨学科混合
【问题意识】<体育学本体问题一句话>
【学科对话】<运动训练学/运动生理/体育社会科学理论> | 缺失
【机制解释】运动学/体育治理含义清晰 / 仅系数或指标（需补）
【贡献】推进/修正 <体育科学认识>
```
