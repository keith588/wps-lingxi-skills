# 《体育学刊》投稿（journal-of-physical-education）

## 期刊定位

华南师范大学主办的 CSSCI 体育学刊物，**以学校体育与体育人文社会学见长**（群众体育、体育政策、体育社会学），同时也收一般体育学稿件。核心要求是**问题意识 + 理论 / 教育规律对话 + 方法规范**。对学校体育稿件，要求超越"教学经验总结"，进入课程 / 教学 / 体质健康的规律与机制；对体育人文社会学稿件，要求与理论实质对话。

## 触发时机

- 题目涉及学校体育（体育课程与教学改革、青少年体质与健康促进、体教融合）、群众体育、体育政策与社会问题
- 数据是教学实验 / 干预、体质监测、调查问卷、政策文本或田野材料
- 文章像"体育教学工作总结"，缺研究问题与机制
- 不确定该投学校体育 / 人文社科向（本刊）还是运动科学 / 竞技向

## 选题偏好

1. **学校体育**：体育课程与教学、教学模式与效果、体教融合、运动技能学习
2. **青少年体质与健康促进**：体质监测、健康行为、运动干预的教育落点
3. **体育人文社会学**：群众体育、体育治理与政策、体育社会问题
4. **一般体育学**：与上述相关的训练 / 运动科学（需规范）

## 方法（教育 / 社科为主，自然科学并收）

- 教育 / 干预路径：教学实验与准实验、量表与问卷、课堂观察、混合方法；强调设计与信效度，把效果翻译成**教育 / 健康规律**
- 社科路径：计量、政策 / 文本分析、访谈 / 田野，强调理论对话
- 自然科学路径：体质 / 运动测试需规范设计与伦理
- 描述性体质 / 参与监测若揭示重要教育 / 健康结构可接纳

## 结构与写作

- 引言立**学校体育 / 体育社会科学问题意识**，进入教育学 / 社会科学脉络
- 干预稿：交代设计、对照、测量与统计，结果翻译成教育 / 健康含义
- 社科稿：把材料 / 系数翻译成社会过程与政策机制，不停在"对策建议"
- 文风学理，避免教学经验汇报腔与口号化表达

## 路由防误投

- 本刊在学校体育与体育人文社会学最具特色；这类稿件优先本刊。
- 先比较：纯运动人体科学实验 → `journal-of-tianjin-university-of-sport` / `china-sport-science`；竞技训练应用 → `china-sport-science-and-technology`；强理论体育社会科学 → `journal-of-shanghai-university-of-sport` / `journal-of-sports-research`；思辨理论 → `sports-and-science`；体育史 / 民族传统体育 → `journal-of-chengdu-sport-university`。
- 常见误投：纯教育学稿若与体育本体关系弱，回教育学渠道。正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式 / 流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从华南师范大学 / 期刊官网、官方采编系统或官方 CNKI 页面开始核验，并说明核验日期。
- 检索"《体育学刊》 投稿须知 / 征稿简则 / 作者指南"，以官方页面为准；不要沿用二手代投网站格式
- 核对匿名正文、作者信息页、基金信息、摘要 / 关键词 / 中图分类号、参考文献格式
- 教学 / 干预稿核对伦理与知情同意（尤其涉及未成年人）、数据要求；社科稿核对调查工具
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的学校体育 / 体育社会科学问题意识，而非工作总结
- [ ] 进入教育学 / 社会科学理论脉络
- [ ] 干预稿设计 / 对照 / 测量 / 统计规范，涉未成年人有伦理保障
- [ ] 结果翻译成教育 / 健康 / 社会过程含义
- [ ] 结论回扣理论或教育规律
- [ ] 无教学经验汇报腔与口号化表达

## 高频拒稿雷区

- 体育教学工作总结，缺研究问题与机制
- 干预稿无对照、测量粗糙、伦理缺失
- 社科稿停在对策建议，无理论对话
- 与体育本体关系弱，更像纯教育学
- 文献与格式不规范

## 决策口诀

- "学校体育 / 体教融合 / 青少年体质 + 规律机制" → 本刊对口
- "体育治理 / 群众体育 + 理论对话" → 本刊或《上海体育大学学报》
- "运动人体科学实验" → 《天津体育学院学报》
- "竞技训练应用" → 《中国体育科技》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】学校体育 / 青少年体质健康 / 体育人文社会学
【问题意识】<学校体育/社科问题一句话>
【对话】教育学/社会科学理论 | 缺失
【机制/效果】教育/健康/社会过程清晰 / 仅经验或对策（需补）
【贡献】推进 <教育规律/理论>
```
