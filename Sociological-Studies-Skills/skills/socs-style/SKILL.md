# 学理文风（socs-style）

## 触发时机

- 通读觉得"像政策汇报"或"像方法报告"
- 结尾全是"政府应加强/完善/推进/深化"
- 方法描述长于社会学阐释
- 大词空转，机制不落地

## 目标语域：社会学的学理克制

本刊的好文风**两头都不靠**：既不是政策汇报腔（把研究写成对策建议），也不是纯技术腔（把社会学问题写成模型展示）。每个判断回扣社会过程，每个大词能兑现为机制与边界。

## 对照表（黑名单 → 白名单）

| 问题 | 黑名单 | 白名单 |
|------|--------|--------|
| 政策汇报腔 | "政府应加强、完善、推进、深化……" | "这一机制表明，<结构条件> 是再生产 <不平等> 的关键环节" |
| 纯技术腔 | "本文采用多层事件史模型，AIC 更优……" | "借助事件史模型，本文得以分离出 <社会过程>" |
| 空洞价值 | "具有重要的理论与现实意义" | "提出 <概念>，重排了对 <现象> 的既有理解" |
| 套话贡献 | "丰富了相关研究" | "修正了 <理论判断> 在 <条件> 下的适用性" |
| 大词空转 | "推动社会治理现代化" | 落到具体机制与行动者，不用不可兑现的大词 |
| 系数崇拜 | "X 显著为正（p<0.01)" | "X 对应约 <…> 的机会差距，意味着 <社会后果>" |

## 行文规范

- **一段一概括**：每段首句是概括句，审稿人读首句即可复原逻辑
- 概念全文一致，慎用未界定的大词与生造术语
- 谦逊与边界：经验研究说"提供了证据支持"，不说"证明了"
- 引用材料/系数后必须**回到社会学解释**，不让证据裸奔
- 方法段不长于机制与理论阐释段

## 微型走查示例（结论段去政策腔，数字为示意）

设某流动人口稿结论段如下，逐句诊断并重写：

```
× 原结论段（政策腔 + 空话 + 系数崇拜）：
  本研究具有重要的理论与现实意义。结果显示随迁子女辍学风险显著更高
  （p<0.01），这丰富了相关研究。政府应进一步加强对流动人口子女教育的
  保障，完善相关制度，推进教育公平。
  诊断：① "具有重要意义"=空话；② "丰富了相关研究"=套话贡献；
       ③ "显著更高"=系数崇拜，无社会后果翻译；④ 末句"加强/完善/推进"
       =政策汇报四件套，把社会学写成对策报告。

√ 重写（学理克制，回扣机制）：
  这一辍学风险的城乡落差并非家庭资源差异所能解释，而是"户籍墙"这一
  制度筛选机制的产物：升学通道与户籍的绑定，把暂时的空间流动固化为
  稳定的机会分层。约 1.8 倍的辍学风险差距（示意），意味着同一段义务教育
  在两类制度位置上兑换出迥异的升学前景。本文据此修正"资源补偿即可
  缩小差距"的判断——在制度筛选未松动前，单纯的资源投入难以触及分层的根源。
```

## 一段自检的"裸奔"测试

随机抽 3 段，看每段是否满足：首句是概括句？段内的系数/材料后是否紧跟一句社会学解读？若某段读完只剩"数字"或"故事"而无机制语言，即为证据裸奔，需补解读。

## 自检清单

- [ ] "具有重要意义"类空话命中 = 0
- [ ] 无政策汇报四件套（加强/完善/推进/深化）
- [ ] 无纯炫技段（方法长度 ≤ 解释长度）
- [ ] 每段首句为概括句
- [ ] 系数/材料后都有社会学解读
- [ ] 大词都能兑现为机制/边界

## 审稿人/编辑追问 + 本刊语境修法

《社会学研究》的文风关切落在"学理克制"，对症修法：

- 追问："结论像政策报告，全是对策建议。" → 修法：删去"加强/完善/推进/深化"四件套，落点改为理论判断的改写。
- 追问："方法描述喧宾夺主，社会学阐释太薄。" → 修法：压缩方法段，使其不长于机制与理论阐释段，把篇幅让给社会过程。
- 追问："大词（现代性/治理/转型）满天飞却不落地。" → 修法：每个大词都要兑现为具体机制、行动者与边界，否则删去。
- 追问："系数后面没有解读，材料在裸奔。" → 修法：每处系数/引文后紧跟一句社会学解读，杜绝证据裸奔。
- 追问："用了'证明了'这类过强措辞。" → 修法：经验研究改用"提供了证据支持/提示"，给出边界与谦逊表述。

## 本刊文风的形态锚点（校准用）

通读本刊近年论文，行文上常见的稳定特征（以编辑部最新说明为准）：概念全文一致、首句概括成段、证据后必有阐释、少用感叹与排比、几乎不出现政策动词收尾。把这五条当作"过稿文风"的近似画像，逐段对照即可。

## 反模式

- 通篇对策建议，把社会学写成政策报告
- 模型与拟合指标占据主场，社会过程缺席
- 用排比大词替代论证
- 用"证明了"等过强措辞，越过经验证据的边界

```

【政策腔】有 / 无    【技术腔】有 / 无

【证据裸奔】X 处（系数/材料无解读）
【大词兑现】到位 / 空转 [...]

```
