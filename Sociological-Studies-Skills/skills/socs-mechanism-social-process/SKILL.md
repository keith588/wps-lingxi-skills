# 社会过程与机制（socs-mechanism-social-process）

## 触发时机

- 定量稿停在"X 显著影响 Y"，说不出"为什么"
- 定性稿停在"现场很生动"，但没有清晰机制
- 只能写"我们认为可能是因为……"

## 核心要求：把证据翻译成"社会过程"

社会学的解释不是"系数为正"，而是一条可叙述的**社会过程**：

```
谁（行动者/群体） + 在什么结构条件下 + 经由什么机制（互动/制度/网络/文化）
+ 产生什么社会后果 + 如何再生产或瓦解某种结构
```

四要素缺一不可：**行动者—结构条件—机制—后果**。

## 机制的常见社会学类型（选最贴切的，别硬套）

- **关系/网络机制**：信息、信任、影响经由社会关系传递
- **制度机制**：规则、组织、分类系统如何约束与赋能
- **文化/认知机制**：分类、意义、正当性如何塑造行为
- **互动机制**：面对面或情境互动中的协商与秩序生成
- **累积/再生产机制**：优势/劣势如何随时间累积、代际传递

## 从系数到过程（定量）

- 把效应量翻译成社会后果：多大的机会差距、多高的流动壁垒
- 用中介/分组检验**支持**机制叙事（中介不是机制本身，只是证据）
- 异质性差异要回答"为什么这个结构位置上更强/更弱"

## 从材料到过程（定性）

- 用扩展个案/过程追踪呈现机制如何一步步发生
- 区分**机制**（如何）与**情境**（何处何时），给出边界条件

## 微型走查示例（代际职业流动，数字为示意）

设有定量稿，基于 CGSS 估出父亲为管理/专业技术职业者，其子女进入同类职业的机会约为农民子女的 3.2 倍（示意）；中介分析显示教育解释了约 45% 的关联（示意）。把系数翻译成社会过程：

```
× 只报系数：父职对子职有显著正向影响（OR=3.2，p<0.001），教育中介占 45%。
  —— 停在显著性，无社会过程，审稿人会问"所以呢？这说明了什么社会学问题？"

√ 翻译成社会过程（四要素）：
  行动者：城市中产家庭的父母与子女。
  结构条件：教育分轨 + 优质教育资源与户籍/学区绑定的制度安排。
  机制（累积/再生产 + 文化）：父母将经济与文化资本前置投入子女学业
    （择校、课外培养、升学规划），把家庭优势转译为可继承的教育资格，
    再由劳动力市场对学历的筛选转化为职业位置——优势经由教育这一
    "正当化通道"完成代际传递，且因其"凭本事"外观而更难被质疑。
  社会后果：阶层边界在"机会平等"的话语下被柔性再生产。
  边界条件：在教育扩张快、学历贬值的群体中，中介作用减弱（示意），
    优势转向非正式资本（关系、信息），提示机制随结构位置切换。
  一句话社会过程：中产家庭经由教育把阶层优势"漂白"为可继承的资格，
    使分层在平等外观下自我再生产。
```

注意：中介占 45% 是机制的**证据**，不是机制本身；机制是"资本转译—正当化—再生产"这条可叙述的过程。

## 自检清单

- [ ] 四要素齐全：行动者 / 结构条件 / 机制 / 后果
- [ ] 机制有明确社会学类型，不是泛泛"影响"
- [ ] 定量：效应已翻译成可理解的社会后果
- [ ] 中介/分组只作机制的证据，不冒充机制
- [ ] 给出机制的边界条件（何时成立/失灵）
- [ ] 过程能被一句话复述，且回扣问题意识

## 审稿人追问 + 本刊语境修法

- 追问："中介变量显著就等于证明机制了吗？" → 修法：说明中介只是与机制叙事一致的证据，机制本身是可叙述的社会过程，并补过程证据。
- 追问："异质性结果和你讲的机制对不上。" → 修法：让机制能解释为何某结构位置上更强/更弱；若解释不了，修正机制而非掩盖。
- 追问："这个机制是社会学的，还是心理学/经济学的？" → 修法：明确机制类型（关系/制度/文化/互动/再生产），落到社会结构层面。
- 追问（定性）："凭一两个案例就断言机制普遍成立？" → 修法：用过程追踪呈现机制逐步发生，并交代边界条件与负面案例。
- 追问："机制和开头的问题意识脱节。" → 修法：结尾用一句话社会过程回扣问题意识，形成闭环。

## 反模式

- 把"统计显著"当机制
- 把中介变量等同于机制本身
- 机制叙事与异质性结果互相矛盾
- 只描述现象，回避"如何发生"
- 机制只在讨论部分出现，与经验结果各说各话

## 输出格式

```
【行动者】<谁>
【结构条件】<在什么条件下>
【机制】<关系/制度/文化/互动/再生产> = <…>
【社会后果】<形成/再生产 什么结构>
【边界条件】<何时成立>
【一句话社会过程】<…>
【下一步】socs-concept-building / socs-style
```
