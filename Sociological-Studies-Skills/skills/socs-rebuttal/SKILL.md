# 外审回复（socs-rebuttal）

## 触发时机

- 收到《社会学研究》的修改意见（R&R）
- 需要逐条回应审稿人与编辑
- **正文已据意见修订完成**（未改正文勿先写回复）

> 本刊实行**匿名评审 + 三审定稿制**（官网投稿指南，2026-05-30 核实）：外审多为同行匿名专家，编辑部终审重学术规范。故回复信须**对审稿人匿名意见与编辑意见分别回应**，并保持双盲（回复中勿暴露作者身份）。

## 社会学审稿的高频关切（对症回应）

| 审稿意见类型 | 回应策略 |
|--------------|----------|
| "问题意识不清/像政策评估" | 重述社会学问题，删政策腔；引 `socs-problematic` 的修订 |
| "没和理论对话/文献是清单" | 补理论脉络，明确继承/修正；引 `socs-theory-dialogue` |
| "只有系数没机制" | 补社会过程叙事与异质性解释；引 `socs-mechanism-social-process` |
| "定性材料薄/编码不透明" | 补材料厚度、编码路径、负面案例、反身性；引 `socs-qualitative` |
| "概念没立住" | 补界定、近邻区分、边界；引 `socs-concept-building` |
| "识别不够干净" | 诚实回应稳健性，但说明本文目标是机制而非净效应 |
| "贡献不足" | 明确改写了哪条理论判断，避免"丰富了研究" |

## 回复信结构

```
尊敬的编辑、审稿专家：
感谢宝贵意见。我们已据此全面修订，逐条回应如下（页码对应修订稿）。

【审稿人一】
意见 1：<原文转述>
回应：<做了什么修改 + 位置（第X页）>。我们认为……
意见 2：……

【审稿人二】……
```

## 微型走查示例（流动人口稿的三态回应，数字为示意）

设审稿人一提三条意见，按"照改/部分采纳/保留申辩"三态分别成文：

```
【审稿人一】
意见 1：问题意识偏政策评估，更像"积分落户有没有效"。
回应（已照改）：感谢指正。我们已重写引言（第 2–3 页），将追问从
  "政策净效应"改为"积分制如何把户籍特权重新编码为可竞争的资格，
  从而再生产城市成员资格的分层"，并删去原结论中的政策建议段（原第 19 页）。

意见 2：教育中介只报了系数，机制叙事不足。
回应（部分采纳并说明）：已在第 11–12 页补入"资本转译—正当化—再生产"的
  社会过程叙事，并将中介结果明确界定为机制的"证据"而非机制本身。
  审稿人建议追加结构方程，我们考虑后保留原中介设定，因样本量在
  分组后受限（分组后部分子样本 N 偏小，示意），SEM 估计不稳健，
  故以分组回归 + 过程叙事替代，并在脚注说明这一取舍。

意见 3：建议补做工具变量以"洗干净"内生性。
回应（保留申辩）：我们理解此关切，并已补充稳健性（第 14 页）。但本文目标是
  揭示回报分化的社会机制，而非估计教育的净因果效应；在缺乏可信外生工具时，
  追加弱工具反而削弱可信度。我们已在第 14 页诚实标注识别边界与解读范围。
```

要点：三态分明、每条标页码、不同意处给学理理由而非情绪化对抗。

## 回应原则

- **逐条、不漏**：每条都回应，即使是部分接受也说明理由
- **可定位**：指明修改落在第几页/哪节，方便复核
- **有礼有据**：尊重但不盲从；不同意时给学理理由，不情绪化
- **一致性**：回复信的修改承诺与正文实际改动一致
- 区分"已照改""已部分采纳并说明""保留并申辩"三态

## 自检清单

- [ ] 每条意见都有回应，无遗漏
- [ ] 每处修改可在正文定位（页/节）
- [ ] 不同意处给出学理理由，语气专业
- [ ] 回复承诺与正文改动一致
- [ ] 社会学问题意识/理论对话/机制的核心关切已正面回应
- [ ] 正文确已修订（非空头承诺）

## 本刊语境下的回复禁忌（额外注意）

- **匿名禁忌**：本刊双盲，回复信中勿出现"我们前期工作（作者名，2023）"等暴露身份的表述；自引一律第三人称。
- **不越级、不催稿**：通过官网系统提交回复，勿私下联系编辑或审稿人；进度查询以系统与编辑部最新通知为准。
- **不与三审定稿制对抗**：编辑部终审重学术规范，回复对编辑意见（规范、体例、篇幅）须同样认真，不可只盯审稿人。
- **核心关切优先**：先正面回应"问题意识/理论对话/机制/可信度"四类硬关切，再处理枝节，勿避重就轻。

## 反模式

- 正文没改就先写"已修改"
- 笼统说"已认真考虑"却无具体改动
- 与审稿人争辩态度对立
- 只回应易答的，回避核心质疑
- 回复信中暴露作者身份，破坏双盲

## 输出格式

```
【意见总数】审稿人 X / 共 Y 条
【处理】照改 A 条 / 部分采纳 B 条 / 保留申辩 C 条
【核心关切】问题意识□ 理论对话□ 机制□ 可信度□ 概念□
【正文一致性】已核对 / 待核对
【风险点】<最可能被追问处>
```
