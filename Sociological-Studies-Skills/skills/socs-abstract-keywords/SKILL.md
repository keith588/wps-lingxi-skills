# 摘要与关键词（socs-abstract-keywords）

## 触发时机

- 正文、问题意识、机制/概念已定，准备写摘要
- 摘要写成背景套话，或超字数
- 关键词随手凑，没体现问题意识与机制

## 本刊硬规范（详见 resources/journal-profile.md）

- **中文摘要不超过 200 字**
- **关键词 3–5 个**（中文）
- **英文摘要不超过 150 个单词**
- 中英文标题、摘要、关键词齐全

> 以官网最新《投稿指南》为准；字数是硬约束，别超。

## 摘要四要素（命题前置，不先铺背景）

1. **问题/争议**：本文要回答的社会学问题（1 句，别长篇背景）
2. **数据/方法**：CGSS/CFPS/CLDS 或田野/访谈，一句带过
3. **核心发现/机制**：经由什么社会过程，形成什么结构（重点）
4. **理论贡献**：提出什么概念/修正什么判断（落点）

## 写法对照

| 黑名单（背景腔） | 白名单（命题前置） |
|------------------|--------------------|
| "随着……日益重要，本文研究……" | "本文追问 <机制> 如何在 <条件> 下再生产 <不平等>" |
| "运用先进方法分析了数据" | "基于 CFPS，借助事件史模型……" |
| "丰富了相关研究" | "提出 <概念>，修正了 <理论判断>" |

## 本刊摘要的形态锚点（校准用）

通读《社会学研究》近年"专题研究""论文"两栏的摘要，可归纳出几条惯例（以编辑部最新《投稿指南》为准）：

- **开篇即问题，不铺时代背景**：第一句多直接给出社会学追问或反常经验。
- **机制/概念是重心**：定量稿点出"经由什么社会过程"，定性稿点出"提炼出什么概念"。
- **数据/方法一句带过**；落点是"修正/扩展/限定了哪条理论判断"，少见政策建议。
- 摘要全篇不出现"加强""完善""推进"等政策动词——一旦出现，多半问题意识已偏。

## 微型走查示例（流动人口子女教育，数字为示意）

设有定量稿《户籍墙内外：流动人口子女的教育机会再生产》，基于 CFPS 面板，事件史模型估出随迁子女初中后辍学风险约为本地子女的 1.8 倍（示意）。先写出一版背景腔摘要，再按四要素重写：

```
× 背景腔（被毙版）：
随着我国城镇化进程不断加快，流动人口规模日益庞大，其子女教育问题
日益受到社会各界广泛关注。本文运用 CFPS 数据，采用先进的统计方法，
分析了流动人口子女的教育状况，丰富了相关研究，并提出政策建议。
（≈110 字：3/4 在铺背景，无机制、无概念、有政策腔——典型退稿摘要）

√ 命题前置版：
既有研究多将随迁子女的教育劣势归于家庭资源不足。本文基于 CFPS 面板与
事件史模型提出，劣势更源于"户籍墙"——一套将升学通道与户籍绑定的制度
分类，经由学籍、考试与转学规则层层筛选，把暂时的空间流动转化为稳定的
机会分层。随迁子女初中后辍学风险显著高于本地子女，且不随家庭收入提升
而消弭。本文据此修正"资源补偿即可缩小差距"的判断，提出"制度性筛选"机制。
（≈170 字，≤200：问题→机制/概念→证据→理论改写，无政策腔）

【关键词】流动人口；教育不平等；户籍制度；制度性筛选；CFPS（5 个，含原创概念）
```

## 关键词选取

- 体现：**问题域 + 理论概念 + 方法/数据**（如：教育不平等、文化资本、生命历程、CFPS）
- 3–5 个，避免过宽（"社会""中国"）或与标题完全重复
- 至少 1 个体现理论对话/原创概念

## 自检清单

- [ ] 中文摘要 ≤ 200 字（已计字）
- [ ] 关键词 3–5 个，含理论概念
- [ ] 英文摘要 ≤ 150 词，与中文一致
- [ ] 命题/问题前置，背景不超过 1 句
- [ ] 含机制与贡献，不止"研究了什么"
- [ ] 中英文标题/摘要/关键词齐全

## 编辑/初审追问 + 本刊语境修法

摘要是编辑部初审最先读到的部分，常见追问与修法：

- 追问："摘要读完仍不知道本文的社会学问题是什么。" → 修法：把第一句改成一句明确的追问句，删去所有时代背景铺垫。
- 追问："摘要只说做了回归，没说发现了什么机制。" → 修法：补一句"经由 <社会过程> 形成 <结构后果>"，让机制进摘要。
- 追问："关键词太宽（如'社会''中国''影响'），检索价值低。" → 修法：换成问题域 + 理论概念 + 数据三类词，至少 1 个原创概念。
- 追问："英文摘要与中文摘要不对应，且超 150 词。" → 修法：以中文定稿为准逐句对译并压缩，核心概念给出稳定英译。
- 追问："摘要出现政策建议口吻。" → 修法：删去对策句，落点改为理论判断的改写（参见 `socs-style`）。

## 反模式

- 摘要 2/3 在铺背景
- 只说做了什么，不说发现了什么机制
- 关键词全是宽泛大词或纯方法名
- 超 200 字还不自知
- 关键词与标题逐字重复，浪费检索位
- 英文摘要机器直译，核心概念译名前后不一

## 输出格式

```
【中文摘要】<…>（X 字，≤200）
【关键词】<…>；<…>；<…>（3–5）
【英文摘要】<…>（X words，≤150）
【命题前置】是 / 否
【含机制+贡献】是 / 否
【下一步】socs-submission
```
