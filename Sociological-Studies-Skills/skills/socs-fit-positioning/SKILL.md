# 匹配度与定位（socs-fit-positioning）

## 触发时机

- 不确定稿子是不是"社会学的问题"，还是"贴了社会学标签的政策评估"
- 数据是 CGSS/CFPS/CLDS，但叙事像经济学实证
- 落点写着"政策建议：政府应加强……"
- 投前想先确认对不对口，避免投错门

## 核心判别：社会学问题 vs 政策评估

| 维度 | 社会学问题（对口） | 经济学式政策评估（改投） |
|------|--------------------|--------------------------|
| 落点 | 社会机制、结构、不平等的生产再生产 | 某政策对某结果的净效应 |
| 因果 | 服务于"社会过程如何发生" | 干净识别本身是目的 |
| 理论 | 与社会学理论对话（分层、网络、制度、文化） | 与经济学理论或纯政策评估对话 |
| 解释 | 谁、经由什么机制、形成什么结构 | 弹性/边际效应/福利 |
| 结论 | 推进/修正社会学理解 | 政策含义与可操作建议 |

> 判据：若把"政策"二字删掉、问题就不成立，多半是政策评估，**改投**。若删掉政策、社会机制问题依然成立且更清晰，则对口。

## 对口的选题家族

1. **社会分层与不平等**：教育、职业、收入、代际流动、机会结构
2. **家庭、性别与生命历程**：婚育、家庭策略、性别分工、生命历程
3. **社会网络与社会资本**：关系、信任、组织、集体行动
4. **城乡、流动与治理**：户籍、迁移、社区、基层治理、数字社会
5. **劳动、组织与制度**：劳动过程、组织、制度变迁、文化与认同

## 本刊常见的"擦边但不对口"退稿模式（识别清单）

编辑部初审最常以"问题意识不足/选题不对口"退回的几类稿件——若中两条以上，先回炉或改投：

- **政策评估披社会学外衣**：内核是"某政策对某结果的净效应"，社会机制只是装饰。
- **变量关系堆砌**："X 显著影响 Y"，但说不清 Y 之上有什么社会结构意涵。
- **方法炫技稿**：贡献落在"识别更干净/模型更新"，而非社会过程。
- **田野猎奇稿**：以"发现一个新田野点/新群体"为卖点，无概念无机制。
- **思辨无经验稿**：通篇概念演绎，缺中国经验的实证支撑（注意：这类更可能对口《中国社会科学》而非本刊"专题研究"）。
- **跨学科大命题稿**：制度/文明/治理层的宏大原创理论，更适合 `social-sciences-in-china`。

## 微型走查示例（积分落户政策，数字为示意）

设有稿件《积分落户政策对农民工市民化的影响》，用 DID 估出政策使落户概率上升约 6 个百分点（示意）。跑一遍判别规则：

```
删政策测试：把"积分落户政策"删掉 → "农民工市民化"还成立吗？
  当前写法：问题=政策净效应，删掉政策后问题塌掉 → 倾向政策评估。
落点检查：结论落在"建议放宽积分门槛" → 政策建议，非社会学理解。
理论对话：只与"政策有效性评估"对话，未进社会分层/公民身份理论 → 缺失。
初判：低匹配，倾向改投 economic-research / china-industrial-economics。

→ 若要改造为社会学问题（可对口路径）：
  把追问从"政策有没有效"改为"积分制如何把户籍特权重新编码为
  可量化、可竞争的资格，从而再生产城市内部的等级与排斥"。
  删掉政策后，"城市成员资格的分层化"这一问题依然成立且更清晰 → 对口。
  落点改为：修正"积分制=机会均等化"的判断，揭示其分层机制。
```

判据复述：删政策后问题塌掉 = 政策评估，改投；删政策后社会机制问题更清晰 = 对口。

## 自检清单

- [ ] 这是关于**社会机制/结构**的问题，而非"某政策有没有效"
- [ ] 删去"政策"后问题依然成立（否则是政策评估）
- [ ] 能指出与哪一支社会学理论对话
- [ ] 数据/材料服务于揭示社会过程，不是为识别而识别
- [ ] 结论推进社会学理解，而非给操作建议
- [ ] 不靠"新田野点/新群体"本身充当贡献

## 反模式

- 把"政策评估 + 显著系数"当社会学贡献
- 用"社会"二字做包装，内核仍是净效应估计
- 选题过于琐碎细分，无结构性意涵

## 再路由

- 落点是经济因果 + 政策含义 → `economic-research` / `china-industrial-economics`
- 跨学科制度/文明层大命题 → `social-sciences-in-china`
- 确认是社会学问题 → 进 `socs-problematic`

## 输出格式

```
【匹配度】高 / 中 / 低
【类型】社会学问题 / 政策评估（建议改投：…）
【删政策测试】问题成立 / 不成立
【选题家族】<分层/家庭/网络/城乡/劳动…>
【对话理论】<一支社会学理论> | 缺失
【下一步】socs-problematic / 改投<刊>
```
