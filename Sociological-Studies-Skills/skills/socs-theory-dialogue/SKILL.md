# 理论对话（socs-theory-dialogue）

## 触发时机

- 文献综述只列近五年实证"谁发现 X 影响 Y"
- 通篇没出现一条社会学理论脉络，只有变量清单
- 自评："我引了很多文献，但没和谁真正对话"

## 文献综述 ≠ 变量清单

本刊看重的是**进入理论脉络**：你的问题坐落在哪条社会学传统里，继承什么、修正什么、推进什么。堆引用不等于对话。

## 三类可对话的理论资源

1. **经典脉络**：分层与阶级、社会团结与失范、科层与权威、资本与场域、互动与意义建构等社会学传统问题
2. **当代中层理论**：社会网络与社会资本、制度与组织、生命历程、文化与认知、关系/过程社会学
3. **本土经验的理论化**：从中国社会过程（关系、单位、家户、流动、基层治理）提炼并与一般理论对话

> 不必为引经典而引经典。对话的标准是：**你的发现改写/扩展/限定了哪条理论判断**。

## 对话的四种动作

- **继承**：沿用某理论的核心机制，应用到新场景并验证边界
- **修正**：指出该理论在某社会条件下失灵或需附加条件
- **整合**：把两条以往各说各话的脉络在你的机制下打通
- **本土化推进**：从中国经验提炼新机制，反哺一般理论（非"中国是又一个样本"）

## 微型走查示例（关系与求职，数字为示意）

设有 CGSS 稿，发现"通过关系求职者"的初职地位并不普遍高于市场求职者，且在体制内外分化明显（示意）。把变量清单式综述改造成理论对话：

```
× 变量清单式综述（被批"没对话"）：
  （张三，2019）发现关系影响求职；（李四，2021）发现关系影响收入；
  （王五，2022）发现关系影响满意度……本文进一步检验关系的作用。
  —— 只是文献流水账，未进入任何理论脉络。

√ 进入理论脉络（源流—分歧—未决—你的动作）：
  源流：格兰诺维特"弱关系优势"与边燕杰"强关系/人情"传统，对关系
    如何传递资源给出两套机制（信息 vs 人情/义务）。
  分歧：弱关系传信息、强关系传人情，孰主导随制度环境而变。
  未决问题：在体制内外分割的劳动力市场中，关系的"货币"是否同质？
  你的动作（修正 + 本土化推进）：本文提出关系在体制内传递的是
    "正当性背书"而非单纯信息或人情，从而修正"强弱关系二分"在
    中国分割市场中的适用边界——这是从中国经验反哺一般理论，
    而非"中国是关系研究的又一个样本"。
```

要点：综述按"源流/分歧/未决"组织，结尾明确你的动作落在继承/修正/整合/本土化推进的哪一种。

## 审稿人追问 + 本刊语境修法

- 追问："文献综述是变量清单，没看到你和谁对话。" → 修法：按源流—分歧—未决问题重组，结尾点明本文的理论动作。
- 追问："只引了近五年实证，缺经典与当代理论脉络。" → 修法：补经典脉络与中层理论，让问题坐落在传统中。
- 追问："这是把西方理论拿来验证，还是真有推进？" → 修法：说清继承/修正/整合/本土化推进的具体一种，避免"中国版的某理论"。
- 追问："只讲中国特殊国情，拒绝一般化。" → 修法：把中国经验提炼为可与一般理论对话的机制/边界，而非画地为牢。
- 追问："综述与后文机制脱节，理论开头摆一摆。" → 修法：让综述里的未决问题正是后文机制要回答的问题，首尾呼应。

## 自检清单

- [ ] 综述按**源流/分歧/未决问题**组织，不是按年份罗列
- [ ] 至少与一条社会学理论脉络实质对话
- [ ] 说清继承/修正/整合/推进中的哪一种
- [ ] 经典与当代文献都有，不只近五年实证
- [ ] 中国经验进入理论（改写机制/边界），不是采样地

## 反模式

- "（A，2020；B，2021；C，2022）发现……"流水账
- 只引方法文献，不引理论文献
- 把西方理论当真理验证，或只讲特殊国情拒绝一般化
- 综述与后文机制脱节，理论"开头摆一摆"

## 输出格式

```
【对话脉络】<分层/网络/制度/文化/行动…>
【既有判断与分歧】<…>
【你的动作】继承 / 修正 / 整合 / 本土化推进
【未决问题】本文要回答的 <…>
【本土理论化】中国经验如何进入理论：<…>
【下一步】socs-quantitative / socs-qualitative
```
