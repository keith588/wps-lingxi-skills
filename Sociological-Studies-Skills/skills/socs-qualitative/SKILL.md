# 定性路径（socs-qualitative）

## 触发时机

- 材料是田野民族志、深度访谈、扩展个案、历史比较
- 稿子读起来像"讲了个好故事"，但没分析抓手
- 审稿担心："材料够吗、编码透明吗、能不能信"

## 立场：材料厚度 + 分析深度 + 可信度

本刊接纳定性传统，但门槛不低于定量。**复述故事不是分析；从材料中提炼概念/机制、与理论往返，才是。**

## 方法路数（明确你站在哪一支）

| 路数 | 核心动作 | 产出 |
|------|----------|------|
| 田野民族志 | 长期在场、深描、参与观察 | 对社会过程的厚描述与机制 |
| 深度访谈 | 半结构、理论抽样、饱和 | 意义建构与类型学 |
| 扩展个案法 | 用个案对话/修正既有理论 | 理论的延伸与修正 |
| 扎根理论 | 开放/主轴/选择性编码 | 自下而上的概念与命题 |
| 历史比较 | 过程追踪、比较案例 | 机制与因果叙事 |

## 可信度（trustworthiness）四要件

- **材料厚度**：在场时长、访谈数、文档来源，足以支撑论断
- **编码透明**：如何从原始材料到概念，可被追溯（不是"我感觉")
- **反身性**：研究者位置、进入方式、对材料的影响有交代
- **饱和与负面案例**：是否到饱和，是否处理了不符合的个案

## 从材料到概念（本刊最看重）

- 不停在"他们说了什么"，要问"这揭示了什么社会机制"
- 用 `socs-concept-building` 提炼概念，用 `socs-theory-dialogue` 与理论往返
- 引用材料要**典型且足量**，避免孤证支撑大判断

## 微型走查示例（村庄养老田野，数字为示意）

设有扩展个案稿，作者在华北某村蹲点 10 个月，访谈老人与子女 36 人（示意），观察到子女"出钱不出力"、由邻里与村集体补位的照料安排。跑一遍可信度与提炼流程：

```
方法路数：扩展个案法（用本案对话/修正"家庭养老 vs 社会养老"二分）。
材料厚度：在场 10 个月（含农忙/农闲两季）；正式访谈 36 人；村志、
  低保与互助养老台账若干（示意）—— 足以支撑机制论断，非孤证。
编码透明：原始田野笔记 → 开放编码"代际责任的再分配"→ 主轴编码
  "经济供养与日常照料的分离"→ 选择性编码定为核心范畴（路径可追溯）。
反身性：作者为本地人，进入便利但需警惕"想当然"，故对反常安排刻意追问。
饱和与负面案例：访至约第 30 人新主题不再出现（饱和，示意）；
  3 户由子女亲自照料的"反例"用于划定机制边界。
从材料到概念：不停在"老人说子女很孝顺"，提炼出
  「照料的去家庭化」——经济供养仍归子女、日常照料外溢到邻里与集体的
  一种代际责任再分配过程（接 socs-concept-building）。
伦理：全部化名，知情同意，敏感家庭纠纷做模糊化处理。
```

由此从"讲了个村庄养老的故事"升级为"提炼出一条可迁移、可与理论对话的机制"。

## 审稿人追问 + 本刊语境修法

- 追问："访谈 N 太小/在场太短，结论撑得起吗？" → 修法：交代抽样逻辑与饱和判据，说明厚度如何匹配论断范围，不外推过度。
- 追问："从材料到概念是怎么来的，编码像黑箱。" → 修法：呈现开放—主轴—选择性编码链条或扩展个案的对话路径，使推断可追溯。
- 追问："这只是复述受访者说法，分析在哪？" → 修法：在引用材料后立即给出社会学解读，提炼概念/机制，杜绝证据裸奔。
- 追问："研究者身份会不会影响材料？" → 修法：补反身性段落，交代进入方式、位置性及对材料解读的影响。
- 追问："以个别金句支撑普遍结论。" → 修法：用典型且足量材料例示，并报告负面案例的处理。

## 自检清单

- [ ] 明确方法路数（民族志/访谈/扩展个案/扎根/历史比较）
- [ ] 材料厚度足以支撑论断（时长/样本/来源交代清楚）
- [ ] 从材料到概念的编码过程可追溯、透明
- [ ] 有反身性与负面案例处理
- [ ] 已从故事提炼出概念/机制，而非复述
- [ ] 匿名与伦理（化名、知情同意）已处理

## 反模式

- 故事复述，无概念无机制
- 用个别金句支撑普遍结论（孤证）
- 编码黑箱，无法追溯
- 把"新田野点"当贡献本身
- 忽略反身性与研究伦理

```
【可信度】编码透明□ 反身性□ 饱和□ 负面案例□
【已提炼概念/机制】<…>（→ socs-concept-building）
【伦理】化名□ 知情同意□

```
