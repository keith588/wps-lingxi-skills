# 《社会学研究》期刊 AI 投稿技能包

面向《社会学研究》（中国社会科学院社会学研究所主办）的投稿指引，覆盖从问题意识到返修的全流程。该刊为 CSSCI 来源期刊、国家社科基金资助期刊，以"学术水平为取稿的唯一标准"。

## 适用场景

- 为投向《社会学研究》的中文稿件提供选题、理论、方法、写作风格的校准
- 帮助作者理解该刊偏好：机制解释、社会学想象力、不卑不亢的学术表达
- 路由到具体环节的技能文件

## 该刊核心偏好

- **问题意识**：从经验中提炼研究问题，不是现象描述
- **理论对话**：与社会学理论对话，不是"贴标签"式引用
- **方法论**：定性要求深描与反身性，定量要求社会学解释而非计量竞赛
- **写作风格**：学术语体，消灭政策报告调和纯技术调

## 子技能目录

| 子技能 | 功能 |
|--------|------|
| socs-problematic | 构建社会学问题意识 |
| socs-theory-dialogue | 进入经典与当代社会学理论对话 |
| socs-qualitative | 定性方法指引（民族志/访谈/扎根理论） |
| socs-quantitative | 定量方法指引（回归/多层模型/社会学解释） |
| socs-mechanism-social-process | 社会过程与机制分析 |
| socs-concept-building | 概念与机制提炼 |
| socs-style | 学术语体规范 |
| socs-abstract-keywords | 摘要与关键词写法 |
| socs-fit-positioning | 选题与刊物匹配 |
| socs-submission | 投稿准备 |
| socs-rebuttal | 返修回复 |
| socs-workflow | 全流程调度 |
